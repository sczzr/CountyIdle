# CountyIdle

Godot 4 + C# 原型项目，核心是“人口 -> 职业 -> 精英 -> 探险 -> 反哺”的县域飞轮。

## 快速运行

1. 使用 Godot `4.2+` 打开 `project.godot`
2. 确认本机安装 `.NET SDK 8`
3. 运行（`F5`）

## 当前可玩能力

- 每秒推进 1 游戏分钟，每 60 分钟进行一次小时结算
- 人口、岗位、产业建筑、工具覆盖联动
- 科研突破（T1/T2/T3）与产能倍率成长
- 精英繁育、自动探险、装备掉落、郡县动态事件
- 双布局 UI（Legacy/Figma）与存档/读档
