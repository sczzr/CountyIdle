# Chinese Seamless Tile Atlas

This folder contains Godot atlas textures and TileSet resources grouped by house/environment.

## Files
- `chinese_house_atlas_256.png`
- `chinese_env_atlas_256.png`
- `ChineseHouseTileSet.tres`
- `ChineseEnvTileSet.tres`
- `chinese_house_atlas_256.json`
- `chinese_env_atlas_256.json`

## Usage
1. Open TileMap in Godot 4.2+
2. Assign `ChineseHouseTileSet.tres` or `ChineseEnvTileSet.tres` to TileSet
3. Paint tiles directly from atlas cells

## Notes
- Tile size: 256x256
- Atlas grid: 4 columns x 3 rows
- All source textures are seamless and edge-validated
