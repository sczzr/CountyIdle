# 人物设计稿（十角色精修版）

## 设计目标
- 解决原稿中 场景单调、服装单调的问题，统一升级为可直接用于立绘、卡面、剧情插图的角色设计。
- 每个角色都包含：**服装设计 + 动作设计 + 场景设计**，并且绑定到 CountyIdle 的核心飞轮环节。
- 风格基调：东方古风县域修行世界，写实比例，细节可落地，不使用夸张西幻元素。

## 统一美术基准（十角色共用）
- **时代质感**：古风县城 + 山野修行，材质以棉麻、丝绸、皮革、铜铁为主。
- **服装层次**：内衬（功能）+ 外衣（身份）+ 配件（职业）三层结构，避免一件袍子走天下。
- **动作结构**：每个角色至少包含待机动作、职业动作、情绪动作，便于后续做动画拆分。
- **场景结构**：前景（道具）+ 中景（角色）+ 远景（地貌/建筑）+ 天气光影，保证画面不空。
- **色彩规则**：职业主色 + 功能辅色 + 点缀高光，避免全员同色同明度。

## 01｜柳观澜（县卫剑修）
- **飞轮环节**：共建共护 / 英雄冒险
- **服装设计**：
  - 月白贴身箭袖内衬，方便高速拔剑。
  - 墨青交领长袍外搭半披甲，肩部有鳞片护片，防切割。
  - 深棕皮护臂与护腿，银丝云雷纹腰封，挂铜制县卫令牌。
- **动作设计**：
  - 待机：左手扶鞘，右手拇指轻叩剑镡，呼吸稳定。
  - 职业动作：前踏半步后挑—压—收三连剑式。
  - 情绪动作：警戒时侧身回望，披风下摆被风掀起。
- **场景设计**：
  - 清晨城墙哨台，雾气压在护城河上。
  - 近景放烽火台木栏与磨损旗帜；远景有县城屋脊与群山。
  - 冷色晨光从侧后方照亮剑刃，形成高对比轮廓。

## 02｜沈青黛（回春医修）
- **飞轮环节**：人口繁衍 / 优秀后代培育
- **服装设计**：
  - 杏白对襟短褙，袖口有可收束绑带，便于急救操作。
  - 青灰长裙外加防污围裳，衣摆绣细小草药纹样。
  - 腰侧悬三层药囊、银针匣、折叠药册，足穿软底布履。
- **动作设计**：
  - 待机：低头翻阅脉案，手指无意识整理针包。
  - 职业动作：半跪把脉后迅速落针，动作轻且准。
  - 情绪动作：听到病患好转时轻松一笑，肩线放松。
- **场景设计**：
  - 雨后药圃长廊，木格窗外有潮湿竹影。
  - 近景摆放药碾、陶罐、蒸汽药炉；中景为诊台与屏风。
  - 暖黄灯光与冷蓝雨光交织，突出治愈感。

## 03｜霍砚辞（机关工匠）
- **飞轮环节**：产业涌现 / 装备打造
- **服装设计**：
  - 靛蓝短打内搭耐磨麻布，袖口做铜扣固定。
  - 深褐皮围裙带多层工具插槽，胸前有防火金属片。
  - 单目护镜可上翻，腰挂卡尺、刻刀、小扳手与齿轮盒。
- **动作设计**：
  - 待机：指尖快速转动铜齿轮，目光专注。
  - 职业动作：蹲位锁扣、旋紧传动轴、拍击启动核心。
  - 情绪动作：调试成功后抬眉吹去手背铁屑。
- **场景设计**：
  - 地火工坊内部，火光映照在铜管与机关臂上。
  - 近景散落蓝图和半成品弩机；远景有吊臂与升降平台。
  - 橙红炉火做主光，顶部冷色天窗做补光，工业层次更强。

## 04｜温知遥（灵植师）
- **飞轮环节**：人口繁衍 / 产业涌现 / 科技涌现
- **服装设计**：
  - 苔绿宽袖上衣配米白束口裤，便于弯腰与攀爬。
  - 藤编腰封内嵌小型水囊，背负竹框苗箱。
  - 袖边绣叶脉纹，耳后常别一枚干花标本。
- **动作设计**：
  - 待机：轻触叶片观察露珠和纹理。
  - 职业动作：撒灵露、扶苗、记录生长参数三步连贯。
  - 情绪动作：发现异变灵草时迅速蹲下，眼神发亮。
- **场景设计**：
  - 薄暮梯田灵圃，层层田埂延伸至山雾中。
  - 前景有木牌标记与灌溉沟渠；中景有温室棚与风铃驱虫。
  - 金橙夕光落在作物边缘，形成温暖的生命感。

## 05｜裴惊雪（弓骑斥候）
- **飞轮环节**：英雄冒险 / 郡县共护
- **服装设计**：
  - 黑红拼接骑装，内层贴身软甲减少拉弓阻力。
  - 右肩加固护肩，左臂护臂加厚防弓弦反弹。
  - 斜背双箭囊（穿甲箭/讯号箭分仓），披短斗篷防风沙。
- **动作设计**：
  - 待机：半蹲压低重心，耳听远处动静。
  - 职业动作：翻身上马、回身拉满弓、松弦后立刻转移。
  - 情绪动作：追踪成功后向同伴打手势示意静默。
- **场景设计**：
  - 黄昏荒原风口，低矮灌木与乱石形成天然掩体。
  - 远景有边哨烽烟与迁徙兽群剪影。
  - 天空紫灰渐变，地面扬尘形成速度线，强化机动感。

## 06｜许归尘（星盘阵师）
- **飞轮环节**：科技涌现 / 郡县防护
- **服装设计**：
  - 玄青长袍分层裁片，衣摆内衬印几何阵纹。
  - 金线勾勒星轨图案，胸前挂可折叠微型星盘。
  - 指节佩戴导灵戒，腰间悬刻符石板与粉笔囊。
- **动作设计**：
  - 待机：抬手拨动星盘刻度，低声演算。
  - 职业动作：单指点地、双手展开、阵纹沿地面扩散。
  - 情绪动作：推演受阻时皱眉停笔，重新校正角度。
- **场景设计**：
  - 夜色观星台，石地刻有旧阵残痕。
  - 前景摆天文仪、沙盘与灯盏；中景为阵纹光环包围角色。
  - 主光来自月色与阵纹冷光，形成神秘理性氛围。

## 07｜唐曜（矿冶铸匠）
- **飞轮环节**：产业涌现 / 装备打造
- **服装设计**：
  - 深灰耐火内衫与加厚皮甲，肩胸处有防火铆钉。
  - 手戴双层隔热手套，腿部绑金属护板防飞溅。
  - 颈部挂矿石成色镜，腰挂锤头更换套件。
- **动作设计**：
  - 待机：单手抡锤试重，脚步微调站位。
  - 职业动作：升温观察、三连重锤、淬火定型。
  - 情绪动作：锻件成功后用手背擦汗并长呼气。
- **场景设计**：
  - 熔炉矿井入口联动工坊，热浪可视化。
  - 近景有矿车、焦炭、铁砧；远景有升降井架与火星雨。
  - 炉火高亮与暗部矿壁形成强烈明暗，突出力量感。

## 08｜卢念白（县学讲师）
- **飞轮环节**：科技涌现 / 职业分化与转职
- **服装设计**：
  - 竹青儒衫外搭浅灰半臂，领口简洁稳重。
  - 白玉束发簪与墨色腰带，随身携便携卷轴筒。
  - 袖口内衬耐墨布料，避免书写时沾污外层衣物。
- **动作设计**：
  - 待机：执卷立于案前，手指轻敲卷轴边。
  - 职业动作：展开舆图、执笔划线、指向关键地标讲解。
  - 情绪动作：学生答对时轻点桌面，露出认可神情。
- **场景设计**：
  - 晨光书院讲堂，木梁高窗，尘粒在光柱中可见。
  - 近景有竹简、算盘、地理沙盘；中景为黑漆讲台。
  - 暖色晨光为主，背景保持中性，突出知识中心。

## 09｜江晚棠（商路管事）
- **飞轮环节**：产业涌现 / 郡县互通
- **服装设计**：
  - 绛紫行商长袍搭配深蓝短披肩，显身份也便于出行。
  - 腰间多格账册包与铜钥匙串，腕部系防切护绳。
  - 发饰采用低调金属件，减少奔走时晃动。
- **动作设计**：
  - 待机：单手托账簿，另一手拨动小算盘。
  - 职业动作：挥旗调度车队、核验封签、迅速记账。
  - 情绪动作：行情大涨时眼神凌厉、语速加快。
- **场景设计**：
  - 夜市码头，灯笼倒映在河面，货船靠岸装卸。
  - 前景有木箱、封条、秤杆；远景有市集棚顶与城门。
  - 暖橙灯火与水面冷反光对比，形成繁忙商业氛围。

## 10｜岳听风（巡夜英雄）
- **飞轮环节**：郡县共建共护 / 英雄冒险
- **服装设计**：
  - 深黑防寒长披风，内层暗纹可反光识别队友。
  - 玄铁轻甲分段覆盖胸腹与大腿，兼顾防护与机动。
  - 腰侧配短刀、信号烟丸与折叠提灯，靴底防滑钉纹。
- **动作设计**：
  - 待机：提灯缓行，目光扫视屋檐与巷口。
  - 职业动作：突发警报时转身—拔刀—护民撤离三段动作。
  - 情绪动作：危机解除后收刀入鞘，低头确认伤者情况。
- **场景设计**：
  - 雪夜城巷，屋檐积雪与灯火并存。
  - 前景有脚印、倒落木桶；中景为巡夜路线路牌。
  - 冷蓝雪光与暖黄灯光交错，营造紧张但守护感强的画面。

## 批量出图建议（可直接给 AI 绘图工具）
- 固定前缀：古风县域修行世界，角色设定立绘，写实比例，细节丰富，电影级光影，东方审美。
- 每次只替换三个变量：角色名 + 服装关键词 + 场景关键词，保持风格一致。
- 推荐输出：
  - 立绘：3:4 或 2:3
  - 卡面：4:5
  - 场景叙事图：16:9
- 若要做同角色多状态图，建议在提示词中加：same character, same face, consistent hairstyle。
