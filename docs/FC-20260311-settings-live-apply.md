# 功能卡：机宜卷即时生效分组与交互修复

## 功能信息

- 功能名：机宜卷即时生效分组与交互修复
- 优先级：P2
- 目标版本：2026-03-11
- 关联系统：`SettingsPanel.tscn`、`SettingsPanel.cs`、`MainClientSettings.cs`、`Main.cs`

## 目标与用户收益

- 目标：让音量、分辨率调整即时生效，并在设置面板内分组明确“即时/批复”逻辑；修复下拉框点击时的放大闪烁与对齐问题，并统一下拉框风格。
- 玩家可感知收益（10 分钟内）：调整窗格/音律时无需点击批复即可看到效果，设置分类更清晰，交互更稳定。

## 实现范围

- 包含：音量与分辨率实时应用；设置项分组；下拉框放大闪烁与对齐修复；下拉框主题配色统一。
- 不包含：设置项新增、存档结构变更、玩法逻辑改动。

## 实现拆解

1. 设置面板新增“即时生效/批复生效”分组标签并调整排序。
2. 音量与分辨率改为即时预览应用，批复仅做持久化收录。
3. OptionButton 弹出时锁定悬停缩放并保持放大态，同时对齐弹出列表位置与宽度，并统一下拉框配色。

## 验收标准（可测试）

- [ ] 调整音量或分辨率时立即生效，无需点击批复。
- [ ] 设置面板可见“即时生效/批复生效”分组。
- [ ] 下拉框点击展开时不再出现闪一下再放大的抖动，并保持放大态，且列表与输入框对齐。
- [ ] 下拉框弹出列表底色、字色与当前卷轴风格一致。

## 风险与回滚

- 风险：即时预览影响“作罢”行为预期。
- 回滚方式：移除预览事件绑定，恢复仅批复应用。
