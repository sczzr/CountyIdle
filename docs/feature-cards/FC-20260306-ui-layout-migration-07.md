# 功能卡（Feature Card）

## 功能信息

- 功能名：参考图内容化 UI 二次复原
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`UI`

## 目标与用户收益

- 目标：在上一轮分区复原基础上，进一步贴合新参考图的具体信息内容与卡片式组件样式。
- 玩家可感知收益（10 分钟内）：指标展示更直观、建筑区与告警区信息密度提升、操作条更清晰。

## 实现范围

- 包含：
  - 顶栏改为资源速率展示 + 周期/倒计时进度条
  - 左栏改为“分配矩阵”卡片行样式
  - 中栏改为“建筑总览/事件地图/统计报表”标签与建筑卡组
  - 右栏改为“待办与警报”+彩色日志
  - 底栏改为“速度控制 + 手动存档/设置读档/放弃殖民地”
- 不包含：
  - 新玩法系统实现
  - 建筑卡真实升级逻辑
  - 资源与事件配置表扩展

## 实现拆解

1. 场景层级改造并固定关键节点命名
2. `Main.cs` 改为新路径绑定与新文案刷新
3. 保留既有核心交互：岗位调整、探险开关、存档/读档、重置

## 验收标准（可测试）

- [ ] 5 大区域结构与参考图一致（顶/左/中/右/底）
- [ ] 顶栏显示资源值与每秒变化、周期与结算倒计时
- [ ] 左栏岗位行支持 `- / 数值 / +` 调整且不出现负值
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：节点树重排后路径失配会导致 `GetNode` 失败
- 回滚方式：回退 `scenes/ui/*.tscn` + `scenes/Main.tscn` + `scripts/Main.cs`
