# 功能卡（Feature Card）

## 功能信息

- 功能名：主界面沙盘缩放/平移交互优化
- 优先级：`P1`
- 目标版本：2026-03-11
- 关联系统：主界面沙盘、县镇地图视图（`CountyTownMapViewSystem`）、世界/外域视图（`StrategicMapViewSystem`）

## 目标与用户收益

- 目标：放大时以鼠标为中心，支持键盘平移，同时保持地图区与外层菜单之间的焦点导航可达。
- 玩家可感知收益（10 分钟内）：滚轮放大更精准，WASD/方向键可快速浏览地形（WASD/方向键逻辑反向），地图区获得焦点后仍可用上下左右切回左栏/底栏菜单。

## 实现范围

- 包含：滚轮缩放锚点跟随鼠标、WASD/方向键平移（逻辑反向）、地图区与外层菜单的显式焦点桥接、世界/外域地图联动、归中复位。
- 不包含：拖拽平移。

## 实现拆解

1. 增加地图平移偏移量并接入绘制与选中逻辑。
2. 滚轮缩放时调整偏移，保持鼠标下方点稳定。
3. 在 `_Process` 中读取键盘输入实现平移。
4. 为地图页签 / 调度按钮与左栏 / 底栏菜单补焦点邻接，避免焦点困在中央地图区。

## 验收标准（可测试）

- [ ] 滚轮放大/缩小时，以鼠标为中心缩放。
- [ ] 使用 `WASD` 与 `↑↓←→` 可平移沙盘（逻辑反向）。
- [ ] 键盘方向键不会误触 Godot 默认焦点跳转。
- [ ] 手柄 / 焦点导航在地图区获得焦点后，仍能切回左栏或底栏外层菜单。
- [ ] 点击“归中”后缩放与平移恢复初始状态。
- [ ] 点击“归中”后触发一次地图刷新。
- [ ] 选中地块与地图交互仍正常。

## 风险与回滚

- 风险：平移无边界限制可能导致暂时移出可见区。
- 回滚方式：移除 `_panOffset` 与鼠标缩放锚点调整逻辑。
