# 功能卡（Feature Card）

## 功能信息

- 功能名：中式庭院贴图化主题（二次迭代）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`UI`

## 目标与用户收益

- 目标：在已有配色主题基础上，增加木纹与宣纸感纹理，提升界面材质感与风格辨识度。
- 玩家可感知收益（10 分钟内）：面板和按钮更像“木构+纸面”元素，视觉层次更明确。

## 实现范围

- 包含：
  - 新增主题贴图资源（SVG）
  - `Theme` 改用 `StyleBoxTexture` 渲染主要控件
  - 维持现有 UI 交互与布局不变
- 不包含：
  - 玩法系统逻辑改动
  - 新增字体资产
  - 场景结构再拆分

## 实现拆解

1. 设计木纹/纸纹贴图资源与按钮状态贴图
2. 更新 `HanCourtyardTheme.tres` 为贴图风格
3. 保留高对比文字色并做构建验证

## 验收标准（可测试）

- [x] 主题使用贴图样式渲染 `PanelContainer` / `Button` / 日志背景
- [x] 按钮普通/悬停/按下状态材质区分明显
- [x] 文字可读性保持稳定
- [x] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：纹理细节过密导致信息噪声上升。
- 回滚方式：回退 `HanCourtyardTheme.tres` 到纯色版。
