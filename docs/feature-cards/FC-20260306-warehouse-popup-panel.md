# 功能卡（Feature Card）

## 功能信息

- 功能名：仓储管理弹窗（独立 tscn）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`MainUI / WarehousePanel / GameLoop / IndustrySystem`

## 目标与用户收益

- 目标：将仓储管理从内嵌页切换为独立弹窗，并按类别展示库存。
- 玩家可感知收益（10 分钟内）：点击“仓储弹窗”即可看到分门别类的库存数据和快捷操作，操作路径更直观。

## 实现范围

- 包含：
  - 新增 `WarehousePanel.tscn` 与 `WarehousePanel.cs`
  - 新增主界面弹窗接入逻辑 `MainWarehousePanel.cs`
  - 保留地图页签，仓储入口改为弹窗按钮
  - 库存分类展示：基础资源 / 矿石资源 / 材料与产成品
- 不包含：
  - 新增仓库分页与筛选
  - 新增仓库排序规则
  - Figma 布局同款弹窗

## 实现拆解

1. 新建仓储弹窗场景与脚本
2. 接入主界面创建、打开、刷新与操作事件
3. 清理旧内嵌仓库页节点
4. 更新规格文档与状态看板
5. 构建验证

## 验收标准（可测试）

- [ ] 点击“仓储弹窗”可打开独立仓储界面
- [ ] 仓储界面按类别显示库存数据
- [ ] 弹窗内按钮可触发矿仓联建/制工具/扩建工坊/扩建官署
- [ ] 关闭弹窗后主界面可继续正常结算与操作
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：弹窗层级与现有设置弹窗冲突导致交互遮挡。
- 回滚方式：恢复仓储内嵌页签方案，移除弹窗实例化代码。
