# 功能卡（Feature Card）

## 功能信息

- 功能名：周边郡图底图可读性修复（ExtractedTileSet 选片优化）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`地图显示`、`UI`

## 目标与用户收益

- 目标：修复周边郡图底图噪点与重复条纹问题，提升地图主信息可读性。
- 玩家可感知收益（10 分钟内）：路线、轮廓、节点更清楚，底图不再抢视觉焦点。

## 实现范围

- 包含：
  - 周边郡图底图瓦片集合改为地表类 tile
  - 底图索引由简单循环改为哈希分布，降低图案条纹
  - 底图透明度下调，减少干扰
- 不包含：
  - 世界地图与县城地图规则改动
  - 战略地图配置数据结构改动

## 实现拆解

1. 重新选择 `ExtractedTileSet` 的地表瓦片坐标集合。
2. 底图铺设索引改为哈希采样，避免重复图案。
3. 调整底图缩放与透明度，保留线框/路线可读性。

## 验收标准（可测试）

- [ ] 周边郡图底图不再出现高密度楼梯/构件图案
- [ ] 周边郡图的轮廓线与路线可读性提升
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：底图透明度过低时，地图层次感可能下降。
- 回滚方式：回退 `StrategicMapViewSystem.cs` 与本功能卡文件。
