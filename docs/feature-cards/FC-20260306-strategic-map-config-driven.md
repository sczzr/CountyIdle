# 功能卡（Feature Card）

## 功能信息

- 功能名：战略地图配置化（世界图/郡图）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`UI`、`地图显示`

## 目标与用户收益

- 目标：将世界图与郡图从脚本硬编码图形改为读取 `data` JSON 配置渲染。
- 玩家可感知收益（10 分钟内）：后续地图形态可通过配置调整，不必每次改代码重编译。

## 实现范围

- 包含：
  - 新增战略地图配置 JSON（区域、边界、路线、节点）
  - 新增配置模型与加载系统
  - `StrategicMapViewSystem` 改为按配置渲染
- 不包含：
  - 真实地理数据导入工具链
  - 探险玩法数值与事件逻辑联动

## 实现拆解

1. 定义地图配置结构并新增 `strategic_maps.json`
2. 新增配置读取系统（含容错回退）
3. 改造世界/郡图渲染逻辑为数据驱动

## 验收标准（可测试）

- [ ] 世界图与郡图可从 `data/strategic_maps.json` 正常渲染
- [ ] 配置字段变更后，渲染结果随之变化（无需改代码）
- [ ] 配置缺失时可回退默认配置，不导致崩溃
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：配置字段非法导致图形错位或不可见
- 回滚方式：回退配置文件与战略地图系统改动，恢复硬编码绘制
