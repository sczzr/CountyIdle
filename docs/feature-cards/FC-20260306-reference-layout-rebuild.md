# 功能卡（Feature Card）

## 功能信息

- 功能名：参考图 1:1 主界面布局复原
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`UI`

## 目标与用户收益

- 目标：将现有主界面布局与信息排版调整为参考图一致的结构（顶部双行信息、左岗位面板、中部地图标签区、右日志列、底部双区按钮）。
- 玩家可感知收益（10 分钟内）：界面信息分区更接近目标视觉稿，主要操作入口与视觉层次更明确。

## 实现范围

- 包含：
  - 调整 `Main.tscn` 的外边距、三栏宽度和底部条布局
  - 重做 `TopBar` / `JobsPanel` / `JobRow` / `WorldPanel` / `EventLogPanel` / `BottomBar` 的节点层级与样式参数
  - 更新 `Main.cs` 文本映射和节点路径绑定以匹配新布局
- 不包含：
  - 玩法规则、数值和存档结构改动
  - 新增美术资源导入
  - 新系统逻辑

## 实现拆解

1. 先完成文档与命名对齐，定义 UI 结构目标
2. 在子场景中重建对应布局与视觉细节
3. 绑定脚本节点并验证核心交互不回归

## 验收标准（可测试）

- [ ] 主界面分区与参考图一致：顶部/左中右主体/底部
- [ ] 顶部显示人口、资源、战斗信息与“下次结算”倒计时
- [ ] 左侧岗位区支持 `x1/x5/Max` 展示及每行 `- / 数值 / +` 操作
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：节点路径和样式层级变化可能导致 `Main.cs` 绑定失效
- 回滚方式：回退 `scenes/Main.tscn`、`scenes/ui/*.tscn`、`scripts/Main.cs`
