# 功能信息

- 功能名：峰令谱三栏式全景卷轴 UI 优化
- 优先级：P1
- 目标版本：2026-03-10
- 关联系统：宗门组织谱系 / 峰脉协同法旨 / UI

## 目标与用户收益

- 目标：将“峰令谱”弹窗从拥挤列表+详情双栏重构为三栏式全景布局，移除深色内嵌滚动框，提升宽屏阅读性。
- 玩家可感知收益（10 分钟内）：信息层级清晰、峰脉详情与职司导览一眼可读，峰令操作更集中。

## 实现范围

- 包含：左栏九峰名录导航；中栏峰脉纪要+部门卡片；右栏职司导览+批令按钮样式强化；文本直接书写在宣纸底。
- 不包含：峰脉规则/协同结算逻辑变更；新增峰脉数据或额外玩法。

## 实现拆解

1. 重构 `SectOrganizationPanel` UI 结构为三栏式布局并拉宽卷轴可视区域。
2. 新增九峰名录可点击导航与选中态样式。
3. 中栏采用“关键字段 + 部门卡片网格”展示，移除深色内嵌滚动框。
4. 右栏整合职司导览与法旨批令按钮，强化主按钮视觉。

## 验收标准（可测试）

- [ ] 打开【峰令】谱系显示三栏布局，左侧名录可点击切换峰脉。
- [ ] 中栏详情以宣纸直写呈现，无深色内嵌滚动框；部门信息为卡片网格。
- [ ] 右栏职司导览与协同峰令按钮可操作，Esc 提示正常。

## 风险与回滚

- 风险：小分辨率下三栏布局可能显得紧凑，需观察字距与滚动体验。
- 回滚方式：恢复 `SectOrganizationPanel` 原双栏布局与旧版 BuildUi 结构。
