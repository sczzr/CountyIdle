# 功能卡：双地图布局（宗门地图 + 世界地图）

## 功能信息

- 功能名：双地图布局（宗门地图 + 世界地图）
- 优先级：`P1`
- 目标版本：当前迭代（2026-03-08）
- 关联系统：`Main.cs`、`WorldPanel.tscn`、`CountyTownMapViewSystem`、`StrategicMapViewSystem`

## 目标与用户收益

- 目标：将主地图入口收敛为“宗门地图 / 世界地图”两张图，减少视图切换负担。
- 玩家可感知收益（10 分钟内）：打开主界面后，地图入口更聚焦，能在宗门经营与世界形势之间快速切换。

## 实现范围

- 包含：
  - 主地图页签压缩为双地图模式；
  - 县城图入口与文案统一转为宗门地图语义；
  - 世界图入口统一为世界地图语义。
- 不包含：
  - 不重写宗门地图生成器；
  - 不新增第三张地图或独立世界编辑器；
  - 不改仓储弹窗与其余主循环逻辑。

## 实现拆解

1. 在 `Main.cs` 中把主地图切换收敛到宗门/世界两个入口。
2. 在 `WorldPanel.tscn` 中隐藏其余地图页签按钮并更新按钮文案。
3. 在宗门图与世界图视图中统一标题与提示文案。
4. 回填系统规格、功能看板、开发列表与平衡日志。

## 验收标准（可测试）

- [x] 主界面顶部地图入口只显示 `宗门地图 / 世界地图`。
- [x] 宗门图默认打开，缩放按钮继续可用。
- [x] `dotnet build .\Finally.sln` 通过。

## 风险与回滚

- 风险：隐藏原郡图/事件/报表/探险页签后，用户需要改走其它入口查看相关信息。
- 回滚方式：恢复 `WorldPanel.tscn` 对应按钮可见性，并把 `Main.cs` 地图页签切换恢复到旧多页签模式。
