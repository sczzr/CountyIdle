# 功能卡（Feature Card）

## 功能信息

- 功能名：地图页回退（停用 ExtractedTileSet / tilemap 链路）
- 优先级：`P0`
- 目标版本：`v0.x`
- 关联系统：`地图显示`、`UI`

## 目标与用户收益

- 目标：回退不稳定的 tilemap 渲染链路，恢复地图可读性与稳定体验。
- 玩家可感知收益（10 分钟内）：周边郡图不再出现异常重复 tile，县城地图纹理恢复正常。

## 实现范围

- 包含：
  - 县城俯瞰图停用 `ExtractedTileSet` 地表取图，恢复原稳定纹理
  - 周边郡图/天下州域停用 tilemap 底图覆盖，恢复纯底板 + 线框绘制
- 不包含：
  - 地图配置数据结构改动
  - 其他玩法机制改动

## 实现拆解

1. 删除县城图中 `ExtractedTileSet` 取图逻辑。
2. 删除战略地图底图 tile 覆盖逻辑。
3. 构建验证并记录回退结果。

## 验收标准（可测试）

- [ ] 周边郡图不再出现大面积重复 tile 图案
- [ ] 县城俯瞰图使用原稳定纹理并保持可读
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：视觉统一目标暂时后移，需要后续更换可用素材再迭代。
- 回滚方式：如需恢复实验链路，回退到 `map-tabs-and-viewport-06` 之前对应代码版本。
