# 功能卡（Feature Card）

## 功能信息

- 功能名：设置面板书卷化（机宜卷）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`SettingsPanel / ClientSettingsSystem / MainShortcutBindings / PopupPanelBase`

## 目标与用户收益

- 目标：将设置面板统一到“浮云宗书卷子面板”规范，用机宜卷替代现代设置窗。
- 玩家可感知收益（10 分钟内）：打开设置时可在统一的卷轴语境里调整语言、窗格、字样、音律与符令，不再和仓储/存档面板风格割裂。

## 实现范围

- 包含：
  - 为 `SettingsPanel` 增加木轴、宣纸、金边装裱等书卷式外观
  - 将标题、提示、行标签、按钮与快捷键录制提示切到“机宜卷 / 符令”语义
  - 统一 `OptionButton / Button / Slider` 的宣纸墨线主题
  - 保持设置项读写、快捷键录制与冲突交换逻辑不变
- 不包含：
  - 不改客户端设置存储结构
  - 不新增新的设置项或快捷键类型
  - 不改主界面设置入口逻辑

## 实现拆解

1. 重绘 `SettingsPanel.tscn` 的卷轴装饰与文案
2. 在 `SettingsPanel.cs` 中追加宣纸/墨线主题样式
3. 将快捷键录制反馈改成书卷语义
4. 同步 `docs/02/05` 中设置面板的描述
5. 执行 `dotnet build .\\Finally.sln`

## 验收标准（可测试）

- [ ] 设置面板呈现书卷化外观，与仓储账册和留影录风格一致
- [ ] 语言、分辨率、字体大小、主音量与快捷键录制仍可正常交互
- [ ] 录制快捷键时提示语与冲突交换反馈统一到“符令”语义
- [ ] `dotnet build .\\Finally.sln` 通过

## 风险与回滚

- 风险：`OptionButton` 与 `HSlider` 的 Godot 原生控件主题覆盖较依赖运行态表现，需要留意不同窗口尺寸下的可读性。
- 回滚方式：回退 `SettingsPanel.tscn` 与 `SettingsPanel.cs` 的视觉与文案改动，保留设置系统逻辑不变。
