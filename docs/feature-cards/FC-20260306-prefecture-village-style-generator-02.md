# 功能卡（Feature Card）

> 历史兼容说明（2026-03-08）：本文保留旧阶段的“县城 / 郡图 / 郡县 / 州府”等表述以便追溯；在当前项目语义中，请分别按“天衍峰驻地 / 江陵府外域 / 天衍峰经营 / 世界地图”理解。
> 当前对外语义：江陵府外域 / 外域附庸据点链路（保留历史 prefecture 技术名）


## 功能信息

- 功能名：江陵府外域外域主城尺度扩展与高密附庸据点城建
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`UI`

## 目标与用户收益

- 目标：将江陵府外域扩大到“外域主城级”视觉尺度，并显著提升外域主城建筑数量与结构密度，风格参照古代外域主城“内外城 + 坊市 + 主街”布局。
- 玩家可感知收益（10 分钟内）：地图范围更大、附庸据点城建更繁盛，能看到明显的城郭与坊市分区，而非少量孤立建筑点。

## 实现范围

- 包含：
  - 扩大江陵府外域边界与默认显示尺度
  - 外域主城建筑从“少量地标”升级为“地标 + 坊市网格高密节点”
  - 增加外域主城风格主街/关隘/坊巷表达与标签
  - 标签按缩放分级显示，避免远景过密
- 不包含：
  - 天衍峰山门图 2.5D 地图资产替换
  - 建筑点击交互与功能面板

## 实现拆解

1. 重构江陵府外域生成器（边界尺度、建筑密度、道路网格）。
2. 扩展标签结构（最小显示缩放阈值）并接入渲染过滤。
3. 更新系统规格与平衡日志，执行构建验证。

## 验收标准（可测试）

- [ ] 江陵府外域默认视角可见更大地图范围与更完整边界
- [ ] 外域主城建筑节点显著增加（明显高于上一版）
- [ ] 缩小地图时次要标签自动减少，放大后恢复细节标签
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：高密节点与文本可能带来局部遮挡。
- 回滚方式：回退本次 `-02` 生成与标签阈值改动，恢复 `-01` 版本。
