# 功能卡（Feature Card）

## 功能信息

- 功能名：周边郡图开封城尺度扩展与高密城建
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`UI`

## 目标与用户收益

- 目标：将郡图扩大到“州府级”视觉尺度，并显著提升城内建筑数量与结构密度，风格参照古代开封城“内外城+坊市+主街”布局。
- 玩家可感知收益（10 分钟内）：地图范围更大、城建更繁盛，能看到明显的城郭与坊市分区，而非少量孤立建筑点。

## 实现范围

- 包含：
  - 扩大郡图边界与默认显示尺度
  - 城内建筑从“少量地标”升级为“地标+坊市网格高密节点”
  - 增加开封风格主街/城门/坊巷表达与标签
  - 标签按缩放分级显示，避免远景过密
- 不包含：
  - 县城 2.5D 地图资产替换
  - 建筑点击交互与功能面板

## 实现拆解

1. 重构郡图生成器（边界尺度、建筑密度、道路网格）。
2. 扩展标签结构（最小显示缩放阈值）并接入渲染过滤。
3. 更新系统规格与平衡日志，执行构建验证。

## 验收标准（可测试）

- [ ] 郡图默认视角可见更大地图范围与更完整边界
- [ ] 城内建筑节点显著增加（明显高于上一版）
- [ ] 缩小地图时次要标签自动减少，放大后恢复细节标签
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：高密节点与文本可能带来局部遮挡。
- 回滚方式：回退本次 `-02` 生成与标签阈值改动，恢复 `-01` 版本。
