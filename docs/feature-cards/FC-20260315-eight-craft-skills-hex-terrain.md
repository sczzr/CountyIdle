# FC-20260315：八大修仙技艺 + 十六Hex地形 重构

## 功能卡片

### 目标（玩家价值）
将原 4 种凡人职业体系完全替换为 **八大修仙技艺**体系，每个技艺有独立、不重叠的核心价值，形成完整的"资源→制造→应用→研究"修仙生产循环；同时配套设计 **十六种Hex地形**，每种地形对应技艺产出，让地图资源和技艺生产完整结合，给玩家真正的修仙宗门经营体验。

### 飞轮环节
**门人培养 → 技艺分化 → 多元产出 → 研究解锁 → 反哺宗门**

覆盖三层空间：
- 山门内部：灵植、炼丹、炼器、阵法、傀儡
- 外域周边：灵兽、天机、符箓、矿脉采集
- 长期成长：天机解锁、技艺进阶、弟子成长

### 依赖（前置系统）
- 现有 hour-tick 主循环
- 现有资源库存系统 `InventoryRules`
- 现有 `GameState` 框架（只需改字段）
- 现有峰令谱 UI（只需适配八技艺显示）

## 设计规格

### 一、八大修仙技艺定义

| 编号 | 技艺名称 | 类型 | 核心产出 | 独立价值 |
|------|---------|------|---------|----------|
| 1 | **灵植** | 资源生产 | 灵药、灵谷、灵木、灵果 | 基础农业/炼丹材料来源，无可替代 |
| 2 | **灵兽** | 单位培育 | 战斗灵兽、坐骑、灵兽材料 | 提供可成长战斗单位，独特玩法 |
| 3 | **炼丹** | 成长消耗 | 聚气丹、疗伤丹、洗髓丹、破阶丹 | 直接加速弟子成长，提供战斗补给 |
| 4 | **炼器** | 装备制造 | 武器、防具、法宝、丹炉、符板、阵盘、傀儡骨架 | 所有装备和加工半成品的唯一来源 |
| 5 | **符箓** | 战斗法术 | 攻击符、防御符、传送符、搜灵符 | 一次性爆发，应对高危战斗 |
| 6 | **阵法** | 区域建设 | 聚灵阵、防御阵、杀阵、困灵阵 | 宗门防御、灵气汇聚、战场控制 |
| 7 | **傀儡** | 自动化生产 | 采矿傀儡、守卫傀儡、采集傀儡 | 解放弟子，自动化生产和守卫 |
| 8 | **天机** | 科技研究 | 解锁丹方、器谱、阵法、新技艺 | 整个游戏的科技树，解锁所有高级内容 |

### 二、各技艺核心规则

#### 2.1 灵植
```
核心职责：
- 在灵田/灵木林/药谷地形种植，产出各类植物材料
- 不同地形产出不同品类

每小时产出公式：
  基础产出 = 弟子数量 × (1 + 地形肥力 × 0.5) × 聚灵阵加成
  变异概率 = 0.5% × 平均弟子资质
  变异产物价值 = 2~5倍普通产物

消耗：
  每弟子每小时消耗 1 份灵谷（自养）

对应地形：灵田、灵木林、药谷、湖泊灵泉、雪峰冰谷
```

#### 2.2 灵兽
```
核心职责：
- 在兽域地形捕捉野生灵兽
- 培养进化，可分配为：
  • 坐骑 → 提高外务移动速度
  • 战斗 → 跟随真传参与探险
  • 放养 → 稳定产出灵兽材料

成长属性：
  灵兽有血脉、等级、好感度、资质
  繁殖可以出更高资质后代（和灵根培育体系联动）

消耗：
  每只灵兽每旬消耗 灵谷/灵草

对应地形：兽域
```

#### 2.3 炼丹
```
核心职责：
- 天机解锁丹方 → 消耗药材 + 丹炉（炼器出）→ 出丹
- 品质分级：凡品 → 上品 → 仙品 → 神品，品级影响效果

丹药分类：
  • 聚气丹 → 加速弟子修为增长
  • 疗伤丹 → 恢复战斗伤势
  • 洗髓丹 → 提升弟子资质（改投修途需要）
  • 破阶丹 → 帮助突破境界瓶颈

对应地形：温泉雾谷（加速炼丹）、各种灵植地形（材料）
```

#### 2.4 炼器
```
核心职责：
- 天机解锁器谱 → 消耗矿材 + 火焰 → 出装备/半成品
- 品质分级：凡 → 精 → 宝 → 仙，品质影响加成

产出分类：
  • 武器 → 增加攻击力
  • 防具 → 增加防御力
  • 法宝 → 特殊能力
  • 丹炉/符板/阵盘/傀儡骨架 → 其他技艺半成品

消耗：
  各类矿材（采矿/傀儡产出）+ 火焰材料（火山产出）

对应地形：青石矿脉、玄铁矿脉、荒原戈壁、火山熔岩
```

#### 2.5 符箓
```
核心职责：
- 消耗灵木浆 + 符板（炼器出）→ 绘制符箓
- 一次性使用，不占用装备栏

分类：
  • 攻击符 → 爆发伤害
  • 防御符 → 临时护盾
  • 传送符 → 瞬间回宗
  • 搜灵符 → 探测区域灵气点位

特点：
  快速制造，批量生产，战斗消耗品

对应地形：灵木林（灵木浆来源）
```

#### 2.6 阵法
```
核心职责：
- 天机解锁阵法 → 消耗阵盘（炼器出）+ 灵石 → 布置在宗门地图
- 持续提供全局/区域增益，需要灵石维持运转

分类：
  • 聚灵阵 → 增加全宗门灵气回复，提高所有产出
  • 防御阵 → 提高护山防御，减少妖患损失
  • 杀阵 → 妖患攻城时造成大量伤害
  • 困灵阵 → 提高捕获灵兽概率

对应地形：空山灵脉（聚灵最佳）、天险关隘（防御最佳）
```

#### 2.7 傀儡
```
核心职责：
- 消耗傀儡骨架（炼器出）+ 灵石核心 → 制作傀儡
- 傀儡可以派去：
  • 采矿 → 自动产出矿材，不占用弟子名额
  • 采集 → 自动采集植物材料
  • 守卫 → 帮助守卫宗门，减少威胁增长
  • 战斗 → 跟随真传参与探险

特点：
  傀儡不占用弟子人口，真正自动化
  每小时消耗少量灵石维护

对应地形：矿脉（材料来源），炼器出骨架
```

#### 2.8 天机
```
核心职责：
- 弟子研学 → 积累研究点数 → 解锁新内容：
  • 新丹方
  • 新器谱
  • 新阵法
  • 新傀儡图纸
  • 地形资源倍率提升

特点：
  不直接产出，但是解锁所有高级内容
  是整个科技树的核心

对应地形：流沙秘境、坊市废墟（遗迹研究产出点数）
```

### 三、生产关系闭环

```
原材料层：
  灵植 → 灵药、灵木、灵谷
  傀儡采矿 → 矿材

加工层：
  炼丹 → 丹药（需要灵植材料 + 炼器出丹炉）
  炼器 → 装备、丹炉、符板、阵盘、傀儡骨架（需要矿材）
  符箓 → 战斗符箓（需要灵木浆 + 炼器出符板）
  傀儡 → 工作傀儡（需要矿材 + 炼器出骨架 + 灵石核心）
  阵法 → 布置阵法（需要炼器出阵盘 + 灵石）

应用层：
  丹药 → 弟子修炼/战斗补给
  装备 → 真传/弟子/灵兽装备
  符箓 → 战斗使用
  傀儡 → 自动化采集/守卫
  阵法 → 宗门增益/防御
  灵兽 → 战斗/材料产出
  天机 → 解锁所有上面的新内容

完整闭环 ✅
```

### 五、地形属性

每个地形有三个基础属性，影响产出：

| 属性 | 范围 | 作用 |
|------|------|------|
| `QiLevel` | 1-5 | 灵气等级，影响产出品质和概率 |
| `Fertility` | 0-10 | 肥力，影响灵植产出 |
| `MineralRichness` | 0-10 | 矿藏丰富度，影响矿材产出 |
