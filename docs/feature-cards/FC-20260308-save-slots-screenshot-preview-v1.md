# 功能卡（Feature Card）

## 功能信息

- 功能名：多存档面板截图预览（V1）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`SaveSystem / SaveSlotSummary / SaveSlotsPanel / MainSaveSlotsPanel / MainShortcutBindings`

## 目标与用户收益

- 目标：让玩家不仅能读摘要数字，还能直接看到该存档当时的大致画面状态。
- 玩家可感知收益（10 分钟内）：多槽面板里能更快分辨“这是哪个阶段/哪张界面”的存档，降低误读档和误覆盖概率。

## 实现范围

- 包含：
  - 写入槽位后保存当前画面截图
  - 多槽详情区显示截图预览
  - 删除手动槽时同步清理预览文件
- 不包含：
  - SQLite 中二进制存图
  - 历史预览版本对比
  - 云端图片同步

## 实现拆解

1. 为槽位摘要补充预览图路径
2. 在存档成功后生成并保存 PNG 预览
3. 在多槽详情区加载并显示预览图
4. 回写系统规格与开发看板

## 验收标准（可测试）

- [ ] 默认槽、手动槽、自动槽写入后会生成对应 PNG 预览
- [ ] 多槽详情区可显示所选槽位的截图预览
- [ ] 没有预览图时会显示明确占位提示
- [ ] 删除手动槽后，对应预览图也被移除
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：截图文件积累会增加 `user://` 存储占用；部分存档动作若发生在弹窗内，截图可能包含当前 UI。
- 回滚方式：保留多槽与筛选排序功能，只移除预览图写入与显示逻辑。
