# 功能卡（Feature Card）

## 功能信息

- 功能名：战略地图配置合法性检查器
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`地图显示`、`配置加载`

## 目标与用户收益

- 目标：在启动加载 `strategic_maps.json` 时执行结构与字段合法性检查，并输出明确告警日志。
- 玩家可感知收益（10 分钟内）：配置异常可快速定位，不会出现无提示的地图空白或错位。

## 实现范围

- 包含：
  - 点数检查（区域/线段至少满足最小点数）
  - 坐标范围检查（建议区间超界告警）
  - 颜色格式检查（非法颜色回退并告警）
  - 告警日志汇总输出（启动阶段）
- 不包含：
  - 自动修复与重写配置文件
  - 编辑器可视化校验工具

## 实现拆解

1. 在配置加载系统加入统一校验流程
2. 对区域/线段/节点逐项做规则检查并记录问题
3. 在渲染系统颜色解析点位接入校验结果输出

## 验收标准（可测试）

- [ ] 非法颜色、点数不足、坐标超界均有 `GD.PushWarning` 提示
- [ ] 校验后地图仍可渲染（使用回退值）
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：校验规则过严导致合法配置被误报
- 回滚方式：回退 `StrategicMapConfigSystem/StrategicMapViewSystem` 校验相关改动
