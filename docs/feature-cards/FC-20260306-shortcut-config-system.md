# 功能卡（Feature Card）

## 功能信息

- 功能名：快捷键配置系统（可重绑定）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`ClientSettings / SettingsPanel / Main`

## 目标与用户收益

- 目标：提供一套可配置、可保存的快捷键系统。
- 玩家可感知收益（10 分钟内）：可在设置面板直接修改常用操作快捷键，并立即生效。

## 实现范围

- 包含：
  - `ClientSettings` 增加快捷键字段并持久化
  - `SettingsPanel` 增加快捷键配置项（下拉重绑定）
  - `Main` 接入按键触发（打开设置、打开仓储、探险开关、倍速切换、存档、读档、重置）
  - 关键冲突处理（非法值回退默认）
- 不包含：
  - 支持组合键（Ctrl/Shift/Alt）
  - 全量操作都可绑定（本次仅高频核心操作）
  - Figma 布局单独快捷键页面

## 实现拆解

1. 扩展客户端设置模型和 Normalize 校验
2. 扩展设置面板快捷键配置 UI
3. 在主控脚本接入快捷键触发逻辑
4. 同步系统规格和功能看板
5. 构建验证

## 验收标准（可测试）

- [ ] 设置面板可配置并保存各项快捷键
- [ ] 重启后快捷键配置仍生效
- [ ] 快捷键触发对应功能且不影响小时结算
- [ ] 设置弹窗打开时不会误触发游戏快捷键
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：按键冲突导致玩家误触。
- 回滚方式：仅保留默认快捷键并隐藏重绑定 UI。
