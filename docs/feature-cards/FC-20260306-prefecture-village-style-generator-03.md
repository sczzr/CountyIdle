# 功能卡（Feature Card）

## 功能信息

- 功能名：开封城城门/坊市命名 JSON 配置化
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`data`

## 目标与用户收益

- 目标：将郡图中的城门、坊市、地貌与地标命名从代码中抽离到 JSON，支持直接改文案与风格。
- 玩家可感知收益（10 分钟内）：改动数据后重开地图即可看到新的城门/坊市命名，不需改 C# 代码。

## 实现范围

- 包含：
  - 新增 `data/prefecture_city_theme.json`
  - 新增主题配置模型与读取系统
  - 郡图生成器改为读取命名配置并渲染
- 不包含：
  - 地图交互系统
  - 城门功能数值联动

## 实现拆解

1. 新增主题配置结构（城门名、坊市名池、地标名、地貌名）。
2. 实现配置读取与容错回退。
3. 接入 `PrefectureMapGeneratorSystem` 并验证构建。

## 验收标准（可测试）

- [ ] 城门名称可在 `prefecture_city_theme.json` 修改并生效
- [ ] 坊市名称池可配置并在地图建筑标签中体现
- [ ] 配置缺失或非法时可回退默认值
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：非法配置可能造成空标签或重复命名。
- 回滚方式：回退主题配置接入，恢复代码内默认命名。
