# 功能卡：宗门经营全局语义统一（V2）

> 历史兼容说明（2026-03-08）：本文保留旧阶段的“县城 / 郡图 / 郡县 / 州府”等表述以便追溯；在当前项目语义中，请分别按“天衍峰驻地 / 江陵府外域 / 天衍峰经营 / 世界地图”理解。


## 功能信息

- 功能名：宗门经营全局语义统一（V2）
- 优先级：`P1`
- 目标版本：当前迭代（2026-03-08）
- 关联系统：`SectMapSemanticRules`、`IndustrySystem`、`JobProgressionRules`、`MapOperationalLinkSystem`

## 目标与用户收益

- 目标：把宗门语义从地图本体继续扩展到岗位、仓储、研究、资源日志、事件提示与地图状态条。
- 玩家可感知收益（10 分钟内）：切到岗位面板、仓储弹窗或世界地图时，不会再突然看到“学宫 / 官署 / 县城 / 市集”等旧郡县词汇，沉浸感更统一。

## 实现范围

- 包含：
  - 扩展宗门术语映射；
  - 替换全局可见旧建筑/地图文案；
  - 收口世界图标题与隐藏备用 Prefecture 文案。
- 不包含：
  - 不修改岗位容量、资源产量、研究阈值等数值；
  - 不重写隐藏 Prefecture 备用逻辑与底层枚举命名。

## 实现拆解

1. 扩展宗门术语规则，补齐建筑/科技/地图称谓。
2. 替换岗位、仓储、研究、资源日志与事件文本。
3. 替换地图调度提示与世界标题。
4. 回写开发列表与平衡归档。

## 验收标准（可测试）

- [x] 岗位与仓储可见文案切到 `阵材圃 / 傀儡工坊 / 传法院 / 山门坊市 / 庶务殿`。
- [x] 研究突破、资源日志、事件提示与天衍峰山门图调度不再出现 `县城 / 学宫 / 官署 / 市集 / 工坊`。
- [x] `dotnet build .\Finally.sln` 通过。

## 风险与回滚

- 风险：隐藏备用 Prefecture 逻辑仍保留旧结构名，若后续重新开放第三张地图，需要继续补更深层语义映射。
- 回滚方式：保留 `SectMapSemanticRules` 新接口，仅把显示层文本调用退回旧字串即可。
