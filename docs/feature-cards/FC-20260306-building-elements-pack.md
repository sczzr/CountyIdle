# 功能卡（Feature Card）

## 功能信息

- 功能名：中式建筑基础元素拆分包（可拼装）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`美术资源 / 地图搭建`

## 目标与用户收益

- 目标：将中式建筑整体风格拆分为独立可复用的基础元素（屋顶、墙体、门窗、柱、台阶、围栏等）。
- 玩家可感知收益（10 分钟内）：开发可快速拼装更多建筑样式，场景变化更丰富且保持统一风格。

## 实现范围

- 包含：
  - 生成一组透明底建筑元素 PNG
  - 输出元素总览图与映射清单（名称/用途）
  - 保持与现有青瓦木构风格一致
- 不包含：
  - 新增玩法逻辑
  - 新增建筑系统规则
  - 替换现有 UI 主题

## 实现拆解

1. 定义基础元素清单（屋顶/墙/柱/门/台阶/围栏）
2. 批量生成可拼装元素资源
3. 输出预览图和元素映射文档

## 验收标准（可测试）

- [x] 产出不少于 12 个独立基础元素 PNG
- [x] 元素为透明底，可直接叠加拼装
- [x] 输出 `preview_sheet` 与 `manifest` 清单
- [x] 与当前中式贴图风格一致（人工目测）

## 风险与回滚

- 风险：元素透视关系不统一会导致拼装时错位感。
- 回滚方式：删除本次元素目录并保留原有贴图集。
