# 功能卡（Feature Card）

## 功能信息

- 功能名：世界/郡/县城地图切换与标签可点击切页
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`UI`

## 目标与用户收益

- 目标：在中部区域新增三层地图视图（世界地图、郡地图、县城地图），并使相关按钮支持点击切换。
- 玩家可感知收益（10 分钟内）：地图层级更明确，可在同一界面快速查看不同尺度信息。

## 实现范围

- 包含：
  - `WorldPanel` 增加地图切换按钮组与三种地图页面容器
  - 原“建筑总览/事件地图/统计报表”按钮接入页面切换逻辑
  - `Main.cs` 新增按钮状态与页面显隐切换
- 不包含：
  - 地图玩法规则与寻路逻辑
  - 建筑卡/报表的后端数据系统

## 实现拆解

1. 定义双层切换结构（功能页切换 + 地图层级切换）
2. 在 `WorldPanel.tscn` 增加页面节点与按钮命名
3. 在 `Main.cs` 绑定并实现点击切换与按钮高亮

## 验收标准（可测试）

- [ ] “建筑总览 / 事件地图 / 统计报表”按钮点击可切换页面
- [ ] “世界地图 / 郡地图 / 县城地图”按钮点击可切换对应地图视图
- [ ] 切换时按钮选中态与页面显示保持一致
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：按钮路径变更导致脚本绑定失败
- 回滚方式：回退 `scenes/ui/WorldPanel.tscn` 与 `scripts/Main.cs`
