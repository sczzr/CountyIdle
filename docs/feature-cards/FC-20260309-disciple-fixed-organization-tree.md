# 功能卡：弟子谱固定宗门组织树

- 功能名：弟子谱固定宗门组织树
- 任务 ID：`DL-043`
- 目标（玩家价值）：让弟子名册不只是“按当前文本临时分组”，而是稳定地挂到宗门既定组织树上，后续更容易承接执事任命、峰内编制和师承管理。
- 飞轮环节：人口繁衍 → 职司分化 → 反哺宗门
- 依赖：`DiscipleProfile.cs`、`DiscipleRosterSystem.cs`、`DisciplePanel.cs`、`SectOrganizationRules.cs`

## 交付范围

- 左侧名册树从“峰脉 -> 堂口 -> 册级 -> 弟子”进一步细化为“峰脉 -> 堂口 / 机构 -> 条线 / 班序 -> 册级 -> 弟子”；
- 峰脉与堂口优先按固定宗门组织树归类；
- 不同职司会落入更稳定的峰内条线，如 `阵枢营造线 / 护山检修线 / 推演研修线 / 商路采办线`；
- 叶节点点击仍驱动右侧档案刷新；
- 保持筛选、排序和地图联动定位。
