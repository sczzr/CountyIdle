# 功能卡（Feature Card）

> 历史兼容说明（2026-03-08）：本文保留旧阶段的“县城 / 郡图 / 郡县 / 州府”等表述以便追溯；在当前项目语义中，请分别按“天衍峰驻地 / 江陵府外域 / 天衍峰经营 / 世界地图”理解。
> 当前对外语义：江陵府外域 / 外域附庸据点链路（保留历史 prefecture 技术名）


## 功能信息

- 功能名：江陵府外域高倍率缩放与建筑可见性增强
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`UI`

## 目标与用户收益

- 目标：提升江陵府外域最大缩放倍率，并在高倍率下把建筑渲染得更像建筑而不是小点。
- 玩家可感知收益（10 分钟内）：滚轮放大到最大时，能明显看见坊市建筑与地标建筑的分布。

## 实现范围

- 包含：
  - `Prefecture` 模式专属更高最大缩放
  - 江陵府外域建筑节点改为高倍率足迹渲染
  - 规格与平衡日志回写
- 不包含：
  - 世界地图缩放规则改版
  - 建筑点击交互

## 实现拆解

1. 调整江陵府外域视图的模式化缩放上限。
2. 为江陵府外域节点增加建筑类型并按类型渲染。
3. 验证构建并回写文档。

## 验收标准（可测试）

- [ ] 江陵府外域最大缩放明显高于当前版本
- [ ] 放大后可看见坊市建筑与地标建筑足迹
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：高倍率下节点过密造成局部遮挡。
- 回滚方式：回退缩放与节点渲染增强，恢复上一版表现。
