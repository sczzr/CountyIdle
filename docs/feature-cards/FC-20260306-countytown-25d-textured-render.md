# 功能卡（Feature Card）

## 功能信息

- 功能名：县城 2.5D 地图贴图化渲染（中式瓦墙资源接入）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`地图渲染 / 美术资源管线`

## 目标与用户收益

- 目标：把县城地图从“程序化纯色块渲染”升级为“程序化布局 + 中式贴图渲染”。
- 玩家可感知收益（10 分钟内）：县城地图建筑观感更接近中式题材，视觉辨识度更高。

## 实现范围

- 包含：
  - 县城地块（草地/道路/庭院/水域）接入现有中式无缝贴图
  - 县城房屋墙体/屋顶接入现有中式房屋贴图
  - 贴图缺失时保留纯色回退，避免渲染中断
- 不包含：
  - 新增贴图资产生产或美术重绘
  - 房屋点击交互与建筑信息弹窗
  - 世界/郡地图贴图化

## 实现拆解

1. 整理并加载可用中式贴图资源路径
2. 在 `CountyTownMapViewSystem` 中改为纹理+颜色混合绘制
3. 保留回退色块并验证构建通过

## 验收标准（可测试）

- [ ] 县城地块使用中式环境贴图渲染
- [ ] 县城房屋屋顶/墙面使用中式房屋贴图渲染
- [ ] 任一贴图加载失败不导致报错，仍可回退渲染
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：小尺寸 2.5D 面片上的贴图细节可能出现压缩噪点。
- 回滚方式：回退 `CountyTownMapViewSystem.cs` 到纯色渲染版本。
