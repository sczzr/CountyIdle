# 功能卡（Feature Card）

## 功能信息

- 功能名：天下州域地图统一使用 `ExtractedTileSet.tres` 底图
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`地图显示`、`UI`

## 目标与用户收益

- 目标：让天下州域页与县城俯瞰/周边郡图一致，统一瓦片素材链路。
- 玩家可感知收益（10 分钟内）：三张地图切换时底图风格一致，视觉统一感更强。

## 实现范围

- 包含：
  - 世界地图模式（`World`）接入 `ExtractedTileSet` 瓦片底图渲染
  - 保留原有世界地图轮廓、路线、节点绘制与缩放行为
- 不包含：
  - 世界地图配置数据结构调整
  - 地图玩法规则与数值改动

## 实现拆解

1. 扩展战略地图底图绘制逻辑，使 `World` 与 `Prefecture` 共用瓦片底图能力。
2. 按模式设置底图透明度，保证线框与节点可读性。
3. 执行构建验证，确认接口与交互兼容。

## 验收标准（可测试）

- [ ] 天下州域地图可见 `ExtractedTileSet.tres` 瓦片底图
- [ ] 周边郡图既有底图效果不回退
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：底图对比度偏高时可能压低世界轮廓线可读性。
- 回滚方式：回退 `StrategicMapViewSystem.cs` 与本功能卡文件。
