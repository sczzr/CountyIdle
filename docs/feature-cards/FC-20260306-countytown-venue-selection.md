# 功能卡（Feature Card）

> 历史兼容说明（2026-03-08）：本文保留旧阶段的“县城 / 郡图 / 郡县 / 州府”等表述以便追溯；在当前项目语义中，请分别按“天衍峰驻地 / 江陵府外域 / 天衍峰经营 / 世界地图”理解。
> 当前对外语义：天衍峰山门图场所选中 / 弟子活动提示


## 功能信息

- 功能名：宗门弟子可视移动系统（五期：场所选中提示）
- 优先级：`P2`
- 目标版本：`v0.x`
- 关联系统：`CountyTownMapViewSystem / TownActivityAnchorData / Resident walkers`

## 目标与用户收益

- 目标：让实体场所不只是“看得见”，还能被选中查看当前活动状态与可视弟子分布。
- 玩家可感知收益（10 分钟内）：点击傀儡工坊、山门坊市、论道亭等场所时，可以立即看到这里现在是否运转、是否热闹，以及有多少可视弟子正在前往或停留。

## 实现范围

- 包含：
  - 左键选中实体场所、右键取消选中
  - 为选中场所添加视觉高亮
  - 在提示条中显示场所名称、状态与可视弟子统计
  - 地图重建后按场所类型与名称尝试保留当前选中项
- 不包含：
  - 场所详情弹窗
  - 对 `GameState` 或经济结算的反向写入
  - 住宅点击与逐户弟子详情

## 实现拆解

1. 在地图视图层新增场所命中检测与选中状态
2. 为选中场所增加高亮轮廓
3. 基于弟子作息 phase 统计“可视 / 前往中 / 当前状态”
4. 把提示条改为“地图摘要 + 运营提示 + 选中场所详情”
5. 编译验证并补日志

## 验收标准（可测试）

- [ ] 左键点击场所可选中，右键可取消
- [ ] 选中场所后有稳定高亮，不因缩放失真
- [ ] 提示条能显示场所名称、状态与可视弟子统计
- [ ] 地图重建或弟子队列刷新后不报错
- [ ] `dotnet build .\Finally.sln` 或等效编译验证通过

## 风险与回滚

- 风险：小地图场所密集时，命中区域过大可能导致误选。
- 回滚方式：保留场所实体与弟子作息，仅移除点击选中与高亮提示逻辑。
