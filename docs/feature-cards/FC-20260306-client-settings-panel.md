# 功能卡（Feature Card）

## 功能信息

- 功能名：客户端基础设置面板（语言/分辨率/字体大小/声音）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`UI`、`core`、`models`

## 目标与用户收益

- 目标：新增独立“设置”界面，让玩家在运行中调整基础客户端体验参数并持久化保存。
- 玩家可感知收益（10 分钟内）：可在游戏内直接调整语言、窗口分辨率、字体大小与声音，不用退出重开手工改配置。

## 实现范围

- 包含：
  - 新增专门设置弹窗（独立场景与脚本）
  - 支持语言、分辨率、字体大小、主音量四类设置项
  - 设置保存/读取与启动自动应用
  - 主界面底栏新增设置入口（保留存档/读档/重置）
- 不包含：
  - 完整多语言文本翻译资产（仅先接入语言切换能力）
  - 复杂图形特效、按设备自动推荐画质

## 实现拆解

1. 增加客户端设置数据模型与存取模块（`models` + `core`）
2. 新建设置面板场景与 UI 交互脚本（`scenes/ui` + `scripts/ui`）
3. 在主界面挂接设置入口并打通“应用 + 持久化”

## 验收标准（可测试）

- [ ] 可从主界面打开独立设置面板，包含语言、分辨率、字体大小、声音项
- [ ] 点击应用后参数立即生效（窗口尺寸/界面缩放/音量）且不破坏主循环
- [ ] 重新启动后设置仍能生效（持久化读取成功）
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：分辨率与字体缩放组合下，低分辨率可能出现局部 UI 拥挤。
- 回滚方式：移除设置面板入口，恢复 `Main.cs` 旧底栏绑定，并删除新增客户端设置读写调用。
