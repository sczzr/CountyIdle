# 功能卡（Feature Card）

## 功能信息

- 功能名：周边郡图生态与城建可视增强
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`UI`

## 目标与用户收益

- 目标：提升周边郡图的信息密度，补齐自然环境与城内建筑表达，解决“建筑偏少、地貌不明确”问题。
- 玩家可感知收益（10 分钟内）：地图可直接识别森林/湖泊/道路/农田/山脉，并在郡城中心看到建筑分区与核心建筑节点。

## 实现范围

- 包含：
  - 程序化郡图增加生态区域（森林、湖泊、农田、山脉）与说明标签
  - 增加城内建筑节点（县衙/市集/工坊/学宫/仓廪/驿站）可视化
  - 提升聚落与道路数量，提高地图建筑与人类活动密度
- 不包含：
  - 县城 2.5D 地图建筑资产替换
  - 地图点击交互与建筑编辑玩法

## 实现拆解

1. 扩展战略地图数据结构，支持地图文字标签。
2. 增强郡图生成器，产出生态区域、更多道路与城建节点。
3. 扩展渲染器绘制标签与新节点，保持缩放兼容。

## 验收标准（可测试）

- [ ] 周边郡图能稳定看到森林/湖泊/农田/山脉区域表达
- [ ] 城内有可见建筑节点，不再只显示单核心点
- [ ] 道路与聚落数量相较上一版明显增加
- [ ] `dotnet build .\Finally.sln` 可执行（允许记录既有阻塞项）

## 风险与回滚

- 风险：标签过密可能影响可读性。
- 回滚方式：回退本次生成器与渲染增强，恢复上一版郡图生成逻辑。
