# 功能卡（Feature Card）

> 历史兼容说明（2026-03-08）：本文保留旧阶段的“县城 / 郡图 / 郡县 / 州府”等表述以便追溯；在当前项目语义中，请分别按“天衍峰驻地 / 江陵府外域 / 天衍峰经营 / 世界地图”理解。
> 当前对外语义：江陵府外域 / 外域附庸据点链路（保留历史 prefecture 技术名）


## 功能信息

- 功能名：江陵府外域生态与附庸据点城建可视增强
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`UI`

## 目标与用户收益

- 目标：提升江陵府外域的信息密度，补齐自然环境与外域主城建筑表达，解决“建筑偏少、地貌不明确”问题。
- 玩家可感知收益（10 分钟内）：地图可直接识别森林/湖泊/道路/农田/山脉，并在外域主城中心看到建筑分区与核心建筑节点。

## 实现范围

- 包含：
  - 程序化江陵府外域增加生态区域（森林、湖泊、农田、山脉）与说明标签
  - 增加外域主城建筑节点（外事别院/坊市/器坊/讲法阁/库藏/灵驿）可视化
  - 提升附庸据点与道路数量，提高地图建筑与人类活动密度
- 不包含：
  - 天衍峰山门图 2.5D 地图建筑资产替换
  - 地图点击交互与建筑编辑玩法

## 实现拆解

1. 扩展战略地图数据结构，支持地图文字标签。
2. 增强江陵府外域生成器，产出生态区域、更多道路与附庸据点城建节点。
3. 扩展渲染器绘制标签与新节点，保持缩放兼容。

## 验收标准（可测试）

- [ ] 江陵府外域能稳定看到森林/湖泊/农田/山脉区域表达
- [ ] 外域主城有可见建筑节点，不再只显示单核心点
- [ ] 道路与附庸据点数量相较上一版明显增加
- [ ] `dotnet build .\Finally.sln` 可执行（允许记录既有阻塞项）

## 风险与回滚

- 风险：标签过密可能影响可读性。
- 回滚方式：回退本次生成器与渲染增强，恢复上一版江陵府外域生成逻辑。
