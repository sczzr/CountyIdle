# 功能卡（Feature Card）

## 功能信息

- 功能名：多存档槽位复制分支（V1）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`SaveSystem / SaveSlotsPanel / MainSaveSlotsPanel`

## 目标与用户收益

- 目标：让玩家能直接从主存档、自动档或任意手动档分支出新手动槽，而不必先读档再另存。
- 玩家可感知收益（10 分钟内）：想“留一个保险档再继续试验”时，可以在面板里一步完成复制，减少误操作和来回切换。

## 实现范围

- 包含：
  - 多槽面板复制入口
  - 底层槽位复制逻辑
  - 槽位预览图同步复制
- 不包含：
  - 外部文件导出/导入
  - 槽位批量管理
  - 存档冲突合并

## 实现拆解

1. 为多槽面板增加复制按钮与事件
2. 为 `SaveSystem` 增加从已有槽位复制到新手动槽的能力
3. 同步复制截图预览文件
4. 回写系统规格、看板和开发列表

## 验收标准（可测试）

- [ ] 可从默认槽、自动槽、手动槽复制出新手动槽
- [ ] 复制后新槽位可正常读取、重命名、删除
- [ ] 若原槽位有截图预览，新槽位会继承预览图
- [ ] 若目标名称为空或未改名，会自动生成“原名 副本”
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：重复槽位名称可能让列表辨识度下降；截图复制失败时，数据与预览可能短暂不一致。
- 回滚方式：保留现有存档和预览系统，只回退复制按钮与底层复制逻辑。
