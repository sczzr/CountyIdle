# 功能卡：江陵府外域备用视图运营语义统一（V2）

## 功能信息

- 功能名：江陵府外域备用视图运营语义统一（V2）
- 优先级：`P2`
- 目标版本：当前迭代（2026-03-08）
- 关联系统：`SectMapSemanticRules`、`MapOperationalLinkSystem`、`StrategicMapViewSystem`

## 目标与用户收益

- 目标：把隐藏备用外域视图的调度按钮、状态提示、标题 fallback 和采集日志也统一到修仙外域语义。
- 玩家可感知收益（10 分钟内）：即使未来重新启用江陵府外域备用视图，也不会在状态条和日志里看到“驿路 / 乡里 / 郡路”这种与当前修仙世界观不匹配的旧词。

## 实现范围

- 包含：
  - 江陵府外域备用视图调度按钮与状态条文案；
  - 外域视图标题 fallback；
  - 采集日志的通用语义收口。
- 不包含：
  - 不调整外域调度的消耗、收益与数值公式；
  - 不重写 `MapDirectiveAction` 枚举与底层地图模式类型名。

## 实现拆解

1. 扩展外域术语映射。
2. 替换外域调度按钮、提示和日志。
3. 修正 Prefecture 标题 fallback。
4. 回写规格与开发清单。

## 验收标准（可测试）

- [x] 江陵府外域备用视图不再显示 `驿路 / 乡里 / 郡路`。
- [x] 外域运营提示统一改为 `灵道 / 附庸据点 / 抚恤附庸`。
- [x] `dotnet build .\Finally.sln` 通过。

## 风险与回滚

- 风险：若未来外域玩法想走完全不同的阵营语义，需要把当前术语从静态规则提取为主题配置。
- 回滚方式：保留外域地图主题，只将运营提示与日志退回旧字串即可。
