# 功能卡（Feature Card）

## 功能信息

- 功能名：小窗口分辨率 UI 自适应修复
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`UI`

## 目标与用户收益

- 目标：修复缩小窗口时中区卡片越界、跨面板覆盖、挤压异常问题。
- 玩家可感知收益（10 分钟内）：在较小分辨率下布局稳定，不再出现控件互相覆盖。

## 实现范围

- 包含：
  - 调整三栏最小宽度，释放中区可用空间
  - 中区 `WorldPanel` 启用裁剪，建筑卡改为流式换行
  - 关键面板启用内容裁剪避免越界绘制
- 不包含：
  - 新玩法与数据逻辑改动
  - 地图系统真实渲染接入

## 实现拆解

1. 降低左右栏最小宽度并保持中栏优先扩展
2. 将建筑卡容器改为 `HFlowContainer` 自动换行
3. 对左右中面板启用 `clip_contents` 避免绘制溢出

## 验收标准（可测试）

- [ ] 缩小窗口后中区卡片不再覆盖右栏/底栏
- [ ] 建筑卡在中区宽度不足时自动换行
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：内容裁剪后极小分辨率下会出现可视区域截断
- 回滚方式：回退 `scenes/Main.tscn`、`scenes/ui/WorldPanel.tscn`、`scenes/ui/JobsPanel.tscn`、`scenes/ui/EventLogPanel.tscn`
