# 功能卡（Feature Card）

## 功能信息

- 功能名：仓储弹窗视觉重制（库房账册书卷风格）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`WarehousePanel / MainWarehousePanel / MaterialSemanticRules / PopupPanelBase`

## 目标与用户收益

- 目标：将现有仓储弹窗重做为“库房账册”书卷布局，用账册清单和批红调度替代现代背包格子，同时保持现有仓储操作链路不变。
- 玩家可感知收益（10 分钟内）：打开仓储后会看到更贴合修仙宗门气质的账册式物资清单，能快速扫视库存、满溢预警与调度批令。

## 实现范围

- 包含：
  - 将仓储弹窗改为卷轴账册：木轴、宣纸主卷、账册标题与朱砂预警
  - 左侧库存改为“物资图鉴清单”双列卡片，保留 `全览 / 农桑 / 金石 / 百工` 页签切换
  - 物资条目优先显示正式材料图片，辅以挂缀品阶点、名称与整数库存
  - 右侧操作改为“天工造物 / 宗门土木 / 供养法旨”三组批令
  - 保留 `Esc` 关闭、提示条短消息和原有仓储操作事件
- 不包含：
  - 不改仓储容量公式、材料产出公式和小时结算顺序
  - 不新增新的仓储玩法或新的资源字段
  - 不改主界面其它面板布局

## 实现拆解

1. 按设计稿重构 `WarehousePanel.tscn` 的卷轴账册布局
2. 重写 `WarehousePanel.cs` 的账册清单、页签与批令样式
3. 接入正式材料贴图与缺图回退逻辑
4. 同步 `docs/02/05/08` 中仓储 UI 描述
5. 执行 `dotnet build .\\Finally.sln` 并记录结果

## 验收标准（可测试）

- [ ] 仓储弹窗呈现“木轴 + 宣纸 + 账册标题 + 朱砂预警”结构
- [ ] `全览 / 农桑 / 金石 / 百工` 页签可切换，库存数量实时刷新
- [ ] 物资条目以账册卡片形式显示正式图片、品阶挂缀、名称与整数库存
- [ ] 右侧按钮仍可触发矿仓联建、锻制工器、设施扩建与四条 `T0` 链路配置
- [ ] `dotnet build .\\Finally.sln` 通过

## 风险与回滚

- 风险：卷轴装饰节点与账册卡片样式较多，需要确认低分辨率下排版稳定，且账册语义不会削弱功能可读性。
- 回滚方式：回退 `WarehousePanel.tscn` 与 `WarehousePanel.cs` 到上一版卡槽弹窗实现，保留仓储系统逻辑与入口不变。
