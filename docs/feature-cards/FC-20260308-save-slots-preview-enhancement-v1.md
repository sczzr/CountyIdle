# 功能卡（Feature Card）

## 功能信息

- 功能名：存档预览增强（V1：摘要字段 + 面板预览）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`SaveSystem / SqliteMigrationRunner / SqliteSaveRepository / SaveSlotsPanel / GameState`

## 目标与用户收益

- 目标：让多存档面板不只显示“人口/金钱”，而是能更直观地展示每个槽位的经营状态摘要。
- 玩家可感知收益（10 分钟内）：打开多存档面板时，可以更快区分“哪个档是繁荣档、哪个档是高威胁档、哪个档更适合继续推进”。

## 实现范围

- 包含：
  - 扩展 `save_slots` 摘要字段
  - 面板列表与详情展示更多存档预览信息
  - 维持现有多槽与自动档逻辑不变
- 不包含：
  - 存档截图预览
  - 地图缩略图
  - 复杂筛选与搜索

## 实现拆解

1. 扩展 SQLite schema 与槽位元信息字段
2. 写入更多 `GameState` 摘要到 `save_slots`
3. 调整多槽面板列表与详情文案
4. 回写系统规格与看板

## 验收标准（可测试）

- [ ] `save_slots` 能记录并读取更丰富的摘要字段
- [ ] 多槽面板列表可看出槽位日期、科技、威胁或民心差异
- [ ] 多槽面板详情能展示资源/风险/探险等更完整状态
- [ ] 旧数据库可通过迁移补列，不破坏现有槽位
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：摘要字段过多会让面板信息过载；SQLite 增列迁移若处理不当会影响旧库。
- 回滚方式：保留新列但回退 UI 展示，继续使用旧摘要文案。
