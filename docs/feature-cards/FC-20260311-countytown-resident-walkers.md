# 功能卡模板（Feature Card）

## 功能信息

- 功能名：停用天衍峰山门图场所/弟子/道路/水域生成
- 优先级：P1
- 目标版本：2026.03.11
- 关联系统：SectMapViewSystem / CountyTownMapViewSystem / TownMapGeneratorSystem / TownMapSelectionSummary

## 目标与用户收益

- 目标：移除地图上的房屋、人物、道路与水域生成，仅保留纯地面六角格。
- 玩家可感知收益（10 分钟内）：地图更清爽，不再出现房屋/弟子与道路/湖泊干扰。

## 实现范围

- 包含：移除住宅/场所/道路/水域生成、停用弟子可视化与场所选中、更新提示与文档状态。
- 不包含：不改 GameState、经济结算、人口/岗位逻辑与存档结构。

## 实现拆解

1. TownMapGeneratorSystem 去掉住宅/场所/道路/水域生成，地图仅保留纯地面格。
2. CountyTownMapViewSystem 停止绘制/选中场所与弟子，更新交互提示。
3. 更新 02/05/08 与 FC/CP/BL 记录。

## 验收标准（可测试）

- [ ] 天衍峰山门图不再生成房屋/场所/弟子/道路/水域可视对象
- [ ] 地图提示与左侧检视器默认文案与当前行为一致
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：失去场所/弟子可视反馈与道路/湖泊地形提示，弱化地图互动
- 回滚方式：恢复场所/弟子与道路/水域生成与绘制逻辑，重启 selection 与提示链路
