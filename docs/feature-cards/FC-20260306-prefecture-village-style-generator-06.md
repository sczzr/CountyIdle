# 功能卡（Feature Card）

> 历史兼容说明（2026-03-08）：本文保留旧阶段的“县城 / 郡图 / 郡县 / 州府”等表述以便追溯；在当前项目语义中，请分别按“天衍峰驻地 / 江陵府外域 / 天衍峰经营 / 世界地图”理解。
> 当前对外语义：江陵府外域 / 外域附庸据点链路（保留历史 prefecture 技术名）


## 功能信息

- 功能名：江陵府外域主街-坊巷高倍细化
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`PrefectureMapGenerator`、`UI`

## 目标与玩家收益

- 目标：让江陵府外域在高倍缩放下不再只是零散建筑块，而是呈现更接近古代外域主城的主街、横街、里巷与沿街街屋。
- 玩家可感知收益（10 分钟内）：放大江陵府外域时能看到主街两侧连续街屋、坊巷分隔、院落建筑与更密集的主城肌理。

## 实现范围

- 包含：
  - 提升外域主城坊市节点密度
  - 在高倍缩放下补充主街/横街/里巷纹理
  - 将坊市节点细化为沿街屋面、巷内庭院与市肆摊列
- 不包含：
  - 建筑点击交互
  - 经营数值与地图双向联动

## 实现拆解

1. 调整外域主城坊市节点网格密度与主城道路数量
2. 在渲染层叠加整座外域主城的主街与里巷纹理
3. 细化高倍缩放下的坊市与院落绘制表现

## 验收标准（可测试）

- [ ] 江陵府外域高倍缩放下可见连续主街、横街与里巷纹理
- [ ] 坊市节点在最大缩放时明显呈现沿街街屋排布
- [ ] 外域主城建筑密度较上一版进一步提高
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：高倍渲染绘制元素增加，可能影响局部可读性与帧率
- 回滚方式：保留惯性缩放，仅回退新增的主街/里巷纹理与高密坊市细节
