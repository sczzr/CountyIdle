# 功能卡（Feature Card）

## 功能信息

- 功能名：郡图御街-坊巷高倍细化
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`PrefectureMapGenerator`、`UI`

## 目标与玩家收益

- 目标：让郡图在高倍缩放下不再只是零散建筑块，而是呈现更接近古代开封城的御街、横街、里巷与沿街街屋。
- 玩家可感知收益（10 分钟内）：放大郡图时能看到主街两侧连续街屋、坊巷分隔、院落建筑与更密集的城内肌理。

## 实现范围

- 包含：
  - 提升郡城坊市节点密度
  - 在高倍缩放下补充御街/横街/里巷纹理
  - 将坊市节点细化为沿街屋面、巷内庭院与市肆摊列
- 不包含：
  - 建筑点击交互
  - 经营数值与地图双向联动

## 实现拆解

1. 调整郡城坊市节点网格密度与城内道路数量
2. 在渲染层叠加整座郡城的御街与里巷纹理
3. 细化高倍缩放下的坊市与院落绘制表现

## 验收标准（可测试）

- [ ] 郡图高倍缩放下可见连续主街、横街与里巷纹理
- [ ] 坊市节点在最大缩放时明显呈现沿街街屋排布
- [ ] 城内建筑密度较上一版进一步提高
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：高倍渲染绘制元素增加，可能影响局部可读性与帧率
- 回滚方式：保留惯性缩放，仅回退新增的御街/里巷纹理与高密坊市细节
