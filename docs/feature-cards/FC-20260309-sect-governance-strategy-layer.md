# 功能卡：宗主治理中枢（一期：方略层去人头化）

- 功能名：宗主治理中枢（一期：方略层去人头化）
- 任务 ID：`DL-037`
- 目标（玩家价值）：让玩家以宗主身份决定宗门发展方向、法令力度与任务重点，而不是手动配置“多少人干什么活”。
- 飞轮环节：职司分化 → 产业供养 → 传承研修 → 反哺宗门
- 依赖：`SectTaskRules`、`SectTaskSystem`、`TaskPanel`、`JobsPadding`、`GameLoop`

## 交付范围

- 将现有“法旨道数 / 请求人数 / 实派人数”玩家视图改为“治理侧重 / 执事落实态势”；
- 将 `TaskPanel` 对外语义改成“宗主中枢”，按钮改成 `收敛 / 推进 / 鼎力推进 / 恢复均衡`；
- 将 `JobsPadding` 对外语义改为“宗主治理摘要”，说明为“宗主定调、执事自动落实”；
- 保留现有底层折算和双轨内务逻辑，不破坏存档兼容与现有小时结算链路。

## 非目标

- 本轮不落完整门规树；
- 本轮不落人才培养面板、执事任命、季度法令与规则编辑器；
- 本轮不推翻现有 `Farmers / Workers / Merchants / Scholars` 兼容汇总层。
