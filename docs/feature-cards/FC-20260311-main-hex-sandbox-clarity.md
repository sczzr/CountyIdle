# 功能卡（Feature Card）

## 功能信息

- 功能名：主界面六边形沙盘清爽化（密度降低）
- 优先级：`P1`
- 目标版本：2026-03-11
- 关联系统：主界面沙盘、县镇地图视图（`CountyTownMapViewSystem`）

## 目标与用户收益

- 目标：降低首屏沙盘信息密度，提升一眼可读性与“清爽感”。
- 玩家可感知收益（10 分钟内）：地图边界更轻、叠加细节更少，视线更集中在选中地块与操作区。

## 实现范围

- 包含：hex 描边降低、低缩放下语义叠加隐藏、默认缩放微调。
- 不包含：地图生成规则、地块语义、结算与存档。

## 实现拆解

1. 调整 hex 描边透明度与线宽。
2. 低缩放禁用道路/水域语义叠加。
3. 提高默认缩放。

## 验收标准（可测试）

- [ ] 默认进入天衍峰沙盘时，hex 边线更轻，不再压画面。
- [ ] 缩放低于阈值时不显示道路/水域叠加，画面更简洁。
- [ ] 点击地块与地图缩放功能仍正常。

## 风险与回滚

- 风险：叠加细节在低缩放下不可见，可能影响地块语义辨识。
- 回滚方式：恢复 `CountyTownMapViewSystem.cs` 相关常量与阈值。
