# 功能卡（Feature Card）

## 功能信息

- 功能名：按钮“灯笼橙”呼吸高亮动画
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`UI`

## 目标与用户收益

- 目标：为所有可交互按钮增加悬停/聚焦呼吸高亮，强化操作反馈并贴合灯笼色主题。
- 玩家可感知收益（10 分钟内）：鼠标移动到按钮上时，按钮会出现柔和的灯笼橙脉冲高亮。

## 实现范围

- 包含：
  - `Main.cs` 统一遍历并绑定按钮 hover/focus 动画
  - 进入时循环呼吸，离开时平滑复位
  - 增加 Tween 生命周期管理，避免残留动画
- 不包含：
  - 玩法逻辑改动
  - 场景结构变更
  - 新增外部动画资源

## 实现拆解

1. 递归收集界面内全部 `Button`
2. 绑定 `MouseEntered/MouseExited/FocusEntered/FocusExited`
3. 统一管理 pulse tween，并在退出树时清理

## 验收标准（可测试）

- [x] 所有按钮悬停时出现橙色呼吸高亮
- [x] 鼠标离开或失焦时颜色平滑恢复
- [x] 重复悬停不会导致动画叠加卡顿
- [x] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：高亮强度过高会影响文本识别。
- 回滚方式：移除 `BindLanternHoverEffects` 与 pulse tween 逻辑。
