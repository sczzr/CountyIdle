# 功能卡（Feature Card）

> 历史兼容说明（2026-03-08）：本文保留旧阶段的“县城 / 郡图 / 郡县 / 州府”等表述以便追溯；在当前项目语义中，请分别按“天衍峰驻地 / 江陵府外域 / 天衍峰经营 / 世界地图”理解。
> 当前对外语义：江陵府外域 / 外域附庸据点链路（保留历史 prefecture 技术名）


## 功能信息

- 功能名：江陵府外域街区级高倍率与缩放惯性
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`UI`

## 目标与用户收益

- 目标：让江陵府外域在最大缩放时进入街区级观感，并让滚轮缩放带有平滑惯性。
- 玩家可感知收益（10 分钟内）：连续滚轮缩放更顺手，放大到最大时能看到一排排街屋块面与院落布局。

## 实现范围

- 包含：
  - 江陵府外域最大缩放继续提升
  - 江陵府外域缩放改为平滑惯性过渡
  - 高倍率下坊市/附庸据点/地标改为街区级块面渲染
- 不包含：
  - 世界图交互重构
  - 建筑点击进入室内

## 实现拆解

1. 将缩放改为目标值 + 平滑过渡。
2. 扩展江陵府外域高倍率节点渲染。
3. 更新规格与平衡日志并完成构建验证。

## 验收标准（可测试）

- [ ] 滚轮缩放时有明显平滑惯性
- [ ] 江陵府外域最大缩放时可看到街屋块面与院落结构
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：高倍率渲染对象增加，可能降低局部可读性。
- 回滚方式：回退惯性与街区级渲染，恢复上一版高倍率表现。
