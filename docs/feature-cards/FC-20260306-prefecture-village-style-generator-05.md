# 功能卡（Feature Card）

## 功能信息

- 功能名：郡图街区级高倍率与缩放惯性
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`StrategicMap`、`UI`

## 目标与用户收益

- 目标：让郡图在最大缩放时进入街区级观感，并让滚轮缩放带有平滑惯性。
- 玩家可感知收益（10 分钟内）：连续滚轮缩放更顺手，放大到最大时能看到一排排街屋块面与院落布局。

## 实现范围

- 包含：
  - 郡图最大缩放继续提升
  - 郡图缩放改为平滑惯性过渡
  - 高倍率下坊市/聚落/地标改为街区级块面渲染
- 不包含：
  - 世界图交互重构
  - 建筑点击进入室内

## 实现拆解

1. 将缩放改为目标值 + 平滑过渡。
2. 扩展郡图高倍率节点渲染。
3. 更新规格与平衡日志并完成构建验证。

## 验收标准（可测试）

- [ ] 滚轮缩放时有明显平滑惯性
- [ ] 郡图最大缩放时可看到街屋块面与院落结构
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：高倍率渲染对象增加，可能降低局部可读性。
- 回滚方式：回退惯性与街区级渲染，恢复上一版高倍率表现。
