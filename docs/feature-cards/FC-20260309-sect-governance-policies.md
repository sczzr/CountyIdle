# 功能卡：宗主治理中枢（二期：发展方向 / 法令 / 育才）

- 功能名：宗主治理中枢（二期：发展方向 / 法令 / 育才）
- 任务 ID：`DL-038`
- 目标（玩家价值）：让玩家不只“定任务轻重”，还能够真正决定宗门的发展方向、当期法令与育才方略。
- 飞轮环节：职司分化 → 产业供养 → 传承研修 → 人口繁衍 → 反哺宗门
- 依赖：`DL-035`、`DL-037`、`TaskPanel`、`GameLoop`、`EconomySystem`、`PopulationSystem`

## 交付范围

- 新增三层治理选择：`发展方向 / 宗门法令 / 育才方略`；
- 发展方向切换后，自动重排治理条目默认侧重；
- 法令与育才方略接入现有小时结算，提供最小但真实的系统收益；
- 主界面与中枢摘要同步展示当前方向、法令与育才计划。
