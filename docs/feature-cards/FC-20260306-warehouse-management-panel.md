# 功能卡（Feature Card）

## 功能信息

- 功能名：专用仓库管理界面（独立页签）
- 优先级：`P1`
- 目标版本：`v0.x`
- 关联系统：`MainUI / WorldPanel / GameLoop / IndustrySystem / ResourceSystem`

## 目标与用户收益

- 目标：提供专门的仓库管理界面，集中管理仓储与矿材链路。
- 玩家可感知收益（10 分钟内）：无需切换多个卡片即可查看仓储占用、矿产库存、材料状态，并直接执行矿仓联建与工具制造。

## 实现范围

- 包含：
  - 新增“仓库管理”地图页签与独立面板
  - 新增仓储总览（容量、占用率、等级）
  - 新增矿产与材料链状态展示（矿石/锭/复材/部件/建材）
  - 新增面板内快捷操作（矿仓联建、制工具、扩建工坊、扩建官署）
- 不包含：
  - 新增独立仓库子场景文件
  - 新增复杂筛选/排序/分页交互
  - Figma 布局专属仓库页（本次仅 Legacy）

## 实现拆解

1. 更新开发清单并补功能卡/提案
2. 在 `WorldPanel.tscn` 新增仓库页签与面板节点
3. 在 `Main.cs` 接入仓库页签切换、状态刷新与按钮事件
4. 更新系统规格与实现状态文档
5. 构建验证并补平衡日志

## 验收标准（可测试）

- [ ] 顶部出现“仓库管理”页签并可切换
- [ ] 页面实时显示仓储容量、占用、矿产/材料关键库存
- [ ] 面板操作按钮可触发对应逻辑并写日志
- [ ] 不影响原有地图页签与缩放行为
- [ ] `dotnet build .\Finally.sln` 通过

## 风险与回滚

- 风险：页签增多后顶部拥挤，部分分辨率下可读性下降。
- 回滚方式：移除仓库页签和相关 `Main.cs` 绑定逻辑，保留原报表卡片入口。
