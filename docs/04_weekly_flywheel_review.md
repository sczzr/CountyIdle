# CountyIdle 周飞轮审卷

> 本文用于每周审看“浮云宗天衍峰的飞轮是否仍在互相牵引”。  
> 它不是周报模板，也不是流水账。  
> 它只回答三件事：**本周宗门是否健康、哪一环失衡、下周先治哪里。**

## 1. 审卷法

- 建议每周固定审卷一次。
- 单次审卷控制在 `30` 分钟内。
- 只记录会改变下周判断的结论，不记录无用流水。
- 若某项没有新结论，宁可留空，不要虚写。

## 2. 周总判

每次审卷先写总判，不准先写细节后补立场：

- 日期：
- 版本：
- 总判：`稳 / 偏 / 危`
- 本周最重问题一句话：
- 本周最值得保留的一条有效做法：
- 下周第一优先整治项：

裁定：

- `稳`：飞轮仍能互相带动，问题可在现有节奏内修补。
- `偏`：已有一至两环拖后腿，若不处理，将继续放大。
- `危`：核心环节已经互相牵制，必须先止血，不开新枝。

## 3. 八环审项

| 审项 | 必看指标 | 本周要判什么 |
| --- | --- | --- |
| 门人生息与补员 | 净门人增长、住房压力、伤病恢复、通勤损耗 | 门人规模是否仍足以支撑后续职司、产业与外务 |
| 产业供养与灵材采炼 | 关键资源小时净增、工具覆盖率、资源赤字时长 | 是否形成“采集 -> 加工 -> 消耗 -> 再生产”的供养闭环 |
| 传承研修与技艺推演 | 研修积累增速、解锁节奏、关键倍率贡献、路线差异度 | 传承是否仍在推着宗门中长期变强 |
| 职司分流与调配 | 岗位分布稳定性、超编回退频次、优先职司保留效果 | 调配是否清楚可控，是否出现单一路线垄断 |
| 灵根苗子与后辈培养 | 精英新增量、优秀后辈占比、培养投入产出比 | 是否形成“培养后辈 -> 反哺宗门”的长期驱动力 |
| 外务历练与战备 | 推进速度、战损、收益风险比、关键战力缺口 | 是否形成“可冒险、可回本、但不失控”的外务节奏 |
| 装备 / 法器与打造 | 打造成率、品质分布、材料缺口、战备评分变化 | 装备链是否真在服务历练、护山与门人培养 |
| 宗门护持与外域反哺 | 建造推进、威胁均值、袭扰损失、恢复时长、外域稳定度 | 山门是否真的护住了外域，并从外域收回秩序与收益 |

填写裁定时，只写三类结论：

- `立住`：该环本周成立，能继续为别环输血
- `发紧`：该环还能转，但已开始拖慢别环
- `漏气`：该环已失去支撑能力，必须优先处理

## 4. 题材与伦理门禁

每周必须单独审一次题材与底线，不得并入数值结论：

- 是否有旧县城 / 郡县语义漏出
- 是否把门人、附庸、据点写成纯耗材
- 是否出现职业歧视、去人性化或压迫式奖励倾向
- 是否偏离“共同建设、共同受益、共同护持”的底线

本节只允许两种结论：

- `守住`
- `失守`

一旦 `失守`，下周优先级自动上提，不得以“数值更好看”为由压后。

## 6. 下周三令

下周动作最多保留 `3` 项，并按优先级落令：

1.
2.
3.

写法要求：

- 每一项都必须是可执行动作，不写空泛目标。
- 每一项都必须能对应到某一环失衡或某一主阻塞。
- 若本周总判为 `危`，三令中至少两项必须是止血项。

## 7. 判级法

- 八环中有 `2` 项以上判为 `发紧`，总判至少为 `偏`
- 核心供养、门人生息、外务护山中任意 `2` 项判为 `漏气`，总判直接为 `危`
- 若本周没有新增判断空间，只是在修补后遗症，下周不得追加大功能
- 若题材与伦理门禁判为 `失守`，总判至少上调一级
