## 飞轮联动状态

### 1) 门人生息与补员

- 指标：净门人增长、住房压力、伤病恢复、通勤损耗
- 结论：门人规模是否稳定支撑后续职司分化与产业供给

### 2) 产业供养与灵材采炼

- 指标：粮 / 木 / 石 / 金与关键灵材小时净增、工具覆盖率、资源赤字时长
- 结论：是否形成“采集 -> 加工 -> 消耗 -> 再生产”的宗门供养闭环

### 3) 传承研修与技艺推演

- 指标：研究点增速、解锁节奏、关键倍率贡献、路线差异度
- 结论：传承系统是否持续提升宗门中长期效率

### 4) 职司分流与调配

- 指标：岗位分布稳定性、超编回退频次、优先职司保留效果、转任触发率（若已实现）
- 结论：是否存在单一路线垄断或调配体验混乱

### 5) 灵根苗子与后辈培养

- 指标：精英新增量、优秀后辈占比、突变触发率、培养投入产出比
- 结论：是否形成“培养优秀后辈 -> 反哺宗门”的长期驱动力

### 6) 外务历练与战备

- 指标：层数推进速度、战损、历练收益 / 风险比、关键战力缺口
- 结论：是否形成“可冒险但不失控”的成长节奏

### 7) 装备 / 法器与打造

- 指标：打造成功率、品质分布、战备评分增长、材料缺口
- 结论：打造与掉落是否共同服务真传成长与护山战备

### 8) 宗门护持与外域反哺

- 指标：建造推进速度、威胁均值、袭扰损失、恢复时长、外域附庸据点稳定度
- 结论：宗门是否有效护持外域、反哺山门，并提升整体安定与富足

### 9) 叙事与伦理健康

- 指标：人均资源、长期幸福趋势、职业公平性异常项、文案旧语义漏出数
- 结论：是否遵守“无职业歧视、共同建设、共同受益、宗门护持”的设计原则
