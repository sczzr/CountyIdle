# 改动提案（Change Proposal）

## 提案信息

- 标题：世界地图全格二级地图入口打通
- 日期：2026-03-12
- 提案人：Codex
- 变更级别：`L2 机制`

## 改动背景

- 当前问题：世界图虽然已有二级地图占位页，但点击命中仍局限于 `Sites`，大量 world hex 只是背景地貌，无法真正“点进去下一层”。
- 证据（数据/玩家反馈）：用户本轮直接要求“世界地图的每一个单元格都能点击进入下一层地图”；当前运行时代码也只对 `_xianxiaWorldMap.Sites` 做命中判定。

## 改动内容

- 改什么：
  - 将世界图点击逻辑改为 `站点优先、地块回退`；
  - 对未命中站点的 world hex，基于地块原始语义生成格子级二级地图入口数据；
  - 将交互顺序收口为“点击先看左侧详情，再由进入按钮打开下一层地图”；
  - 补齐 `Wilderness` 在左侧检视器和 `SecondaryMapView` 占位页中的模板分支，并在 `SecondaryMapView` 中生成一张格子驱动、与山门沙盘同形的局部 hex 沙盘。
- 不改什么：
  - 不在本轮直接新增独立的野外/遗迹/坊市场景；
  - 不把格子级入口直接接入小时结算、存档或专属事件系统；
  - 不重做世界图渲染框架。
- 影响系统：
  - `StrategicMapViewSystem`
  - `MainSectTileInspector`
  - `MainWorldSitePanel`
  - `docs/02_system_specs.md`
  - `docs/05_feature_inventory.md`
  - `docs/08_development_list.md`

## 预期结果

- 预期提升指标：
  - 世界地图任意 hex 都具备“下一层身份”，不再只剩少数点位可进入；
  - 玩家点击世界格后会先看到左侧详情，再通过进入按钮打开统一二级页，并看到类型、风险、收益与筹备方向；
  - 二级页会按当前格的地貌/水体/奇观/建筑语义生成一张与山门沙盘同形的局部 hex 沙盘，而不是只显示文字占位；
  - `Wilderness` 正式成为可进入的分层玩法入口，而不是背景语义。
- 可接受副作用：
  - 一期仍以模板化入口页为主，格子级入口与真实二级玩法之间还有一段距离；
  - 地块回退生成的语义是规则映射，不是手工精修地名。

## 验证计划

- 验证方式：
  - 在世界地图点击已有站点与普通 hex，确认两者都可进入 `SecondaryMapView`；
  - 确认 `Wilderness` 在左侧检视器和二级页中显示专属文案，而不是默认 fallback；
  - 运行 `dotnet build .\Finally.sln`。
- 观察周期：本轮实现当次验证。
- 成功判定阈值：
  - 世界地图任意 hex 点击后都有下一层入口；
  - 世界地图任意 hex 点击后，左侧检视器都能显示对应详情；
  - 点击进入按钮后，`SecondaryMapView` 能基于当前格生成局部地图；
  - 现有站点入口行为不回退；
  - 文档和看板状态与代码实现一致。
