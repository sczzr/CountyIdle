# 改动提案：门规树（一期：三支门规纲目）

- 任务 ID：`DL-044`
- 标题：给宗主中枢补上“常设门规底盘”
- 改动级别：`L2 机制增强`

## 背景

当前宗主已经可以决定发展方向、法令、育才和季度法令，但这些仍偏“方针”与“季令”。  
如果没有常设门规，宗门缺少稳定的制度底盘，宗主治理仍会显得偏短期。

## 本轮提案

1. 增加 `庶务 / 传功 / 巡山` 三条门规支线；
2. 每条支线同一时间只允许激活一个节点；
3. 门规作为常设章程，持续影响收益、人口、威胁和工器链路；
4. 继续保持最小闭环，不在本轮引入完整解锁条件与树状前置。

## 风险

- “门规树”如果节点太少，容易更像三条选项而不够像完整树；
- 若与季度法令叠得过多，玩家可能难以分辨短期与长期层的区别；
- 如果恢复逻辑不统一，中枢会出现“部分恢复、部分残留”的混乱感。

## 缓解

- 明确将本轮定位为“门规树一期：三支纲目”；
- 文案上强调“门规 = 常设章程，季度法令 = 当季专项”；
- 让“恢复均衡”同时重置三条门规。

## 验收

- 玩家能切换三条门规支线；
- 门规能真实影响经营结果；
- 构建通过。
