# 改动提案：季度法令（宗主治理三期先行）

- 任务 ID：`DL-042`
- 标题：给宗主中枢补上“按季度颁令”的短周期治理层
- 改动级别：`L2 机制增强`

## 背景

当前宗主中枢已经有 `发展方向 / 宗门法令 / 育才方略` 三层，但都偏常态治理。  
如果没有“季度法令”这种更短周期、可阶段性强化的专项指令，宗主对宗门节奏的推动仍然不够有“当季拍板”的感觉。

## 本轮提案

1. 在宗主中枢追加“季度法令”层，作为短周期宏观指令；
2. 法令类型围绕供养、讲法、护山、坊市、百工五个方向展开；
3. 法令效果挂到小时结算、人口链路与工器锻制；
4. 当季度切换时，法令自动失效，避免长期叠加成常驻被动。

## 风险

- 若法令强度过高，可能和“发展方向”抢角色；
- 若法令不自动失效，会逐渐退化为另一层长期被动；
- 若 UI 不明确标出当前季度，玩家可能不理解为何法令会过期。

## 缓解

- 将季度法令控制在“中等力度但更聚焦”的区间；
- 强制季度轮换自动失效；
- 在中枢摘要中同步显示当前季度与当前法令。

## 验收

- 玩家能切换季度法令；
- 法令会随季度轮换自动失效；
- 构建通过。
