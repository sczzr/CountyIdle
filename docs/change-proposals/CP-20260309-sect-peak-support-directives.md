# 改动提案：峰脉协同法旨（JobsList 三期）

- 任务 ID：`DL-041`
- 标题：把“九峰详情”推进成可下发的协同峰令
- 改动级别：`L2 机制增强`

## 背景

当前 `JobsList` 已经可以浏览九峰与附属部门，但这些信息仍偏静态。  
如果玩家只能“看见”组织结构，却不能借此发出宗门协同法旨，那么九峰仍然更像设定集，而不是治理工具。

## 本轮提案

1. 新增“本季协同峰”机制，默认状态为 `诸峰均衡`；
2. 允许宗主在峰脉详情区直接指定某一峰提供本季协同；
3. 协同峰效果挂到小时结算、门人生息与工器锻制，而不是停留在纯文案层；
4. 保持原有宗主中枢与任务系统兼容，不推翻现有治理三层与法旨折算链路。

## 风险

- 峰脉协同与“发展方向 / 法令 / 育才”同时存在时，玩家可能一时难以区分二者分工；
- 若峰脉效果过强，可能掩盖发展方向的存在感；
- 若界面不显示当前协同峰状态，玩家会忘记自己正在偏置哪一峰。

## 缓解

- 把峰脉协同限定为“当前季度 / 当前轮”的单峰偏置，和治理三层形成“宏观方向 + 组织支援”的分工；
- 数值保持在温和加成区间，避免覆盖治理三层的主导性；
- 在峰脉详情与职司摘要里同步显示当前协同峰状态。

## 验收

- 玩家能从 `JobsList` 直接下发协同峰令；
- 协同峰选择后，至少影响一个小时结算数值与一个非结算链路（如人口或工器）；
- 构建通过。
