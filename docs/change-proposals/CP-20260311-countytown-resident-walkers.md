# 改动提案模板（Change Proposal）

## 提案信息

- 标题：停用天衍峰山门图场所/弟子可视化生成
- 日期：2026-03-11
- 提案人：Codex
- 变更级别：L2 机制

## 改动背景

- 当前问题：地图上房屋与人物可视化已不再需要。
- 证据（数据/玩家反馈）：用户明确要求删除地图上房子和人物，并要求彻底移除生成逻辑。

## 改动内容

- 改什么：移除住宅/场所生成与弟子可视化生成；禁用场所/门人选中与提示。
- 不改什么：经济结算、人口/岗位、存档格式、世界地图逻辑。
- 影响系统：TownMapGeneratorSystem / CountyTownMapViewSystem / TownMapSelectionSummary / 文档。

## 预期结果

- 预期提升指标：地图杂讯降低，视野更清晰。
- 可接受副作用：失去场所/弟子可视反馈与地图联动入口。

## 验证计划

- 验证方式：运行主界面天衍峰山门图，确认无房屋/人物；检查提示文案。
- 观察周期：当次启动。
- 成功判定阈值：0 个房屋/场所/弟子可视对象，地图提示不再引导选中。

## 回滚条件

- 触发条件：需要恢复可视化或地图反馈不足。
- 回滚步骤：恢复生成/绘制与选择逻辑，回写文档。
