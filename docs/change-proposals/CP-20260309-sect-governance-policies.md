# 改动提案：宗主治理中枢（二期：发展方向 / 法令 / 育才）

- 任务 ID：`DL-038`
- 标题：把宗主中枢从“单层任务力度”扩展为“三层治理决策”
- 改动级别：`L2 机制增强`

## 背景

在一期里，玩家已经不再直接看到人数配置，但中枢仍主要围绕治理条目轻重展开。  
如果没有“发展方向 / 法令 / 育才”这三层，宗主的决策仍然偏窄，缺少真正的宏观治理感。

## 提案内容

1. 新增 `发展方向`，用于决定宗门资源更偏向供养、研修、护山还是外务；
2. 新增 `宗门法令`，用于影响民心、威胁、贡献与研修；
3. 新增 `育才方略`，用于影响人口增长与不同人才线的回报；
4. 保持底层兼容，不推翻现有小时结算链路。

## 验收

- 玩家能直接在宗主中枢切换三层治理选项；
- 切换后，摘要、日志与小时结算都会出现可感知差异；
- 构建通过。
