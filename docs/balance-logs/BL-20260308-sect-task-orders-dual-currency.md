## 改动摘要

- 改动项：建立“法旨兼容底层”，新增 `贡献点` 并接入宗门内务的 `贡献点 + 灵石` 双轨消耗。
- 改动前：
  - 玩家直接加减四类岗位；
  - 所有建造、锻器、宗门内务调度只看 `灵石` 与材料；
  - 缺少宗门内部流通货币，内外经济边界不清晰。
- 改动后：
  - 玩家通过治理入口发放阵材采炼、阵枢营造、巡山警戒、阵法推演、总坊值守、外事行商等法旨；
  - 系统将法旨自动折算为四类职司；
  - 内务建造、锻器、天衍峰山门图内务调度统一追加 `贡献点` 成本；
  - 外域交易与外务仍只看 `灵石`。

## 复盘

- 有效原因：
  - 保留旧 `Farmers / Workers / Merchants / Scholars` 作为兼容层，使现有系统无需大拆；
  - 任务面板与职司摘要分层后，玩家行为从“调人头”切换为“定方向”，天衍峰经营感更强；
  - 把 `贡献点` 限定在宗门内务场景，能清楚划出内外经济边界。
- 无效原因：
  - 当前任务仍是抽象法旨，尚未细化到弟子个体或具体工单链；
  - 默认法旨分配与贡献点产出速度仍需要后续试玩校准。
- 后续假设：
  - 下一轮可补“任务模板 / 季度法旨方案 / 事件驱动自动回退”；
  - 若玩家仍希望更细粒度管理，可在任务制之上叠加“执事模板”而不是回到直接调岗位。
