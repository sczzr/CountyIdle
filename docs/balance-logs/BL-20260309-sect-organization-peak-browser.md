# 数值平衡日志：宗门组织谱系详情浏览（JobsList 二期）

- 任务 ID：`DL-040`
- 时间：`2026-03-09`

## 改动前

- `JobsList` 虽然已经有“九峰与附属部门”概览，但仍偏像静态摘要；
- 玩家很难顺着四条主职司继续理解每一峰的真实组织结构；
- 《浮云宗-天衍峰》中的处室级信息还没有充分露出到玩家界面。

## 改动后

- `JobsList` 新增“峰脉详情浏览”块，可逐峰切换查看定位、职责、核心机构与处室；
- 点击四条主职司时会自动聚焦推荐峰脉，让职司与峰系的映射更直观；
- 天枢峰、天机峰、天工峰、天权峰、天元峰、天衡峰补充到处室级显示。
- 后续迁移补强后，峰脉详情浏览已改由独立 `SectOrganizationPanel` 卷册承载，主界面左栏只保留 `Tile Inspector`。

## 指标结果

- 新增峰脉详情浏览块：`1`
- 可切换浏览峰系：`9`
- 处室级详情覆盖峰数：`6+`
- 独立卷册入口：`1`（底部 `【峰令】谱系`）
- 构建结果：`dotnet build .\Finally.sln` 通过

## 结论

这一轮仍然属于可视化与可解释性增强，不直接改动产出公式。  
但它显著降低了玩家理解“谁在支撑天衍峰运转”的成本，也为后续做“峰脉支援 / 执事委任 / 峰间协同”提供了自然入口。
