# 数值平衡日志：宗门弟子独立属性界面（弟子谱）

- 任务 ID：`DL-043`
- 时间：`2026-03-09`

## 改动前

- 玩家只能从总人口、职司摘要、宗门组织谱系与地图可视移动推测“谁在做什么”；
- 缺少一个能集中查看弟子年龄、修为、状态与培养建议的独立入口；
- 真传、外务、推演等弟子之间的差异缺乏个体层展示。

## 改动后

- 主地图页签新增“弟子谱”入口，并打开独立弹窗；
- 天衍峰山门图点击可视弟子或实体场所时，可直接联动打开弟子谱并定位代表弟子；
- 弹窗可按真传 / 职司筛选，按修为 / 潜力 / 心境 / 贡献排序；
- 每名弟子可查看年龄、修为、当前差事、居所、关联峰脉、特征、培养建议与八项属性；
- 名册由 `DiscipleRosterSystem` 基于当前经营态势派生生成，不新增结算字段。

## 指标结果

- 新增独立入口：`1`
- 地图联动入口：`弟子点击 + 场所点击`
- 筛选维度：`7`
- 排序维度：`5`
- 个体属性条：`8`
- 构建结果：`dotnet build .\Finally.sln` 通过

## 结论

这一轮仍属于表现与可解释性增强，不直接调整资源、人口或战斗公式。  
但它把“门人总量”拆成了可读的个体名册，有助于玩家理解育才方略、真传结构与当前职司分布，也为后续做“弟子培养 / 任命 / 外派历练”提供了自然承接入口。
