## 改动摘要

- 改动项：宗门实体场所支持选中与状态提示
- 改动前：
  - 场所已经拥有实体建筑表现；
  - 玩家仍需“目测”弟子活动，无法直接确认某个场所当前是否开工或热闹。
- 改动后：
  - 左键可选中场所，右键可取消；
  - 选中场所会获得高亮轮廓；
  - 提示条显示场所名称、类型、当前状态、可视弟子数与前往中人数。

## 结果数据

- 指标 1：场所状态由弟子当前 phase 聚合而成，不新增 `GameState` 字段。
- 指标 2：工作类场所显示“上工路上 / 开工中 / 已收工”，休闲类场所显示“有人前往 / 热闹中 / 清闲中”。
- 指标 3：地图重建时会按“场所类型 + 名称”尝试保留当前选中项。

## 结论

- 是否达到预期：`是`
