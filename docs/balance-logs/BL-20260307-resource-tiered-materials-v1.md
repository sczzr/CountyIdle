## 改动摘要

- 改动项：资源系统分层扩展（V1：T0-T1 原材料与加工链）四期推进
- 改动前：
  - 当前运行版只有抽象仓储与矿产链，`精盐 / 药剂 / 麻布 / 皮革 / 铜锭 / 熟铁` 尚未进入运行时结算
  - 制工具仍只吃 `铁矿`，人口系统也无法感知盐药衣料来源
- 改动后：
  - `GameState` 新增 `T0/T1` 材料库存字段并保留旧档兼容兜底
  - `ResourceSystem` 接入乡里采集、工坊初加工、民生产线、`铜锭/熟铁` 冶铸
  - `IndustrySystem` 制工具优先消耗 `熟铁/铜锭`
  - `PopulationSystem` 读取 `精盐 / 药剂 / 麻布 / 皮革` 影响幸福、康复与衣物覆盖
  - `WarehousePanel` 能直接观测新材料链路
  - `EconomySystem` 不再直产 `木/石`，木料/石料改由材料链提供
  - `T0` 建筑链显式化为四条可扩建链路，并接入仓储面板交互
  - 可见库存统一为整数，小数变化沉淀到隐藏进度池
