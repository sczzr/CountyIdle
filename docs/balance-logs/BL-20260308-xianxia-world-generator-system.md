## 改动摘要

- 改动项：为世界图新增修仙 Hex 世界生成系统，并把结果适配到现有战略地图视图。
- 改动前：世界图为通用配置驱动区域图，缺少灵脉、奇观、宗门候选与灵性资源层。
- 改动后：世界图默认生成 `Biome / Terrain / Water / Cliff / Overlay / Resource / SpiritualZone / Site / Wonder` 八层语义，并显示河流、灵脉、奇观、宗门候选与稀有资源标记。

- 指标 3：战略图适配结果 = `2560` 个 region、`61` 个节点、`31` 条标签，且 `dotnet build .\Finally.sln` 通过
