# CountyIdle Skill 使用指南

## Skill 列表

- `countyidle-system-design`
- `countyidle-feature-delivery`
- `countyidle-balance-lab`
- `countyidle-agent-handoff`

## 场景到 Skill 的映射

- 设计新机制/重构旧规则：`countyidle-system-design`
- 写 C# 功能代码并联动 UI：`countyidle-feature-delivery`
- 调整数值与节奏：`countyidle-balance-lab`
- 多 Agent 交接和并行协作：`countyidle-agent-handoff`

## 推荐工作流

1. 需求评估：`countyidle-system-design`
2. 功能实现：`countyidle-feature-delivery`
3. 数值验证（如需要）：`countyidle-balance-lab`
4. 任务移交：`countyidle-agent-handoff`

## 与现有模板联动

- 功能卡：`docs/templates/feature-card-template.md`
- 改动提案：`docs/templates/change-proposal-template.md`
- 平衡日志：`docs/templates/balance-log-template.md`
- 交接模板：`docs/templates/handoff-template.md`
